# <a name="configure-the-functionality-profile-for-a-sales-representative"></a><span data-ttu-id="d94b1-103"> Konfigurace funkčního profilu pro obchodní zástupce</span><span class="sxs-lookup"><span data-stu-id="d94b1-103">Configure the functionality profile for a sales representative</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d94b1-104">Tato procedura ukazuje, jak konfigurovat nastavení funkce profilu prodejny vztahující se na obchodní zástupce pomocí ukázkových dat společnosti USRT.</span><span class="sxs-lookup"><span data-stu-id="d94b1-104">This procedure demonstrates how to configure a store's functionality profile settings that apply to sales representatives using the USRT demo data company.</span></span>

1. <span data-ttu-id="d94b1-105">Přejděte na Retail and Commerce > Instalace kanálu > Nastavení POS > Profily POS > Funkční profily.</span><span class="sxs-lookup"><span data-stu-id="d94b1-105">Go to Retail and Commerce > Channel setup > POS setup > POS profiles > Functionality profiles.</span></span>
2. <span data-ttu-id="d94b1-106">Klikněte na položku Upravit.</span><span class="sxs-lookup"><span data-stu-id="d94b1-106">Click Edit.</span></span>
3. <span data-ttu-id="d94b1-107">Rozbalte oddíl Funkce.</span><span class="sxs-lookup"><span data-stu-id="d94b1-107">Expand the Functions section.</span></span>
    * <span data-ttu-id="d94b1-108">Můžete použít nastavení profilu funkce ke konfiguraci POS na automatické přidání výchozí prodejní skupiny pokladníka, k vyzvání k zadání prodejních skupin a k požadování prodejních skupin.</span><span class="sxs-lookup"><span data-stu-id="d94b1-108">You can use the functionality profile settings to configure POS to automatically add the cashier's default sales group, to prompt for sales groups, and to require sales groups.</span></span>  
4. <span data-ttu-id="d94b1-109">Poté vyberte možnost v poli Požadavek na výběr prodejního zástupce.</span><span class="sxs-lookup"><span data-stu-id="d94b1-109">In the Prompt for sales representative field, select an option.</span></span>
5. <span data-ttu-id="d94b1-110">Vyberte možnost Ano v poli Vyžadovat prodejního zástupce.</span><span class="sxs-lookup"><span data-stu-id="d94b1-110">Select Yes in the Require sales representative field.</span></span>
