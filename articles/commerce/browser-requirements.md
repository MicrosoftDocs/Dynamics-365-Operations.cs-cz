# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Požadavky na prohlížeč pro weby Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

Tento článek pokrývá požadavky prohlížeče na weby Microsoft Dynamics 365 Commerce.

Weby Dynamics 365 Commerce jsou nejlépe podporovány následujícími webovými prohlížeči. Poslední verzi pro každý webový prohlížeč naleznete na webu výrobce softwaru.

- [Microsoft Edge](https://www.microsoft.com/edge) (do aktuální verze minus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (do aktuální verze minus 1)
- [Google Chrome](https://www.google.com/chrome) (do aktuální verze minus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (do aktuální verze minus 1)

> [!NOTE]
> Internet Explorer není podporován. Pokud používáte Internet Explorer, některé funkce nemusí fungovat a uživatelé mohou mít nestandardní prostředí.
