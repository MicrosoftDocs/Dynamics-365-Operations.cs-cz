# <a name="troubleshoot-store-commerce-performance-issues"></a>Odstraňování problémů s výkonem Store Commerce

[!include [banner](../includes/banner.md)]
[!include [banner](../includes/preview-banner.md)]

Tento článek vysvětluje, jak řešit problémy s výkonem v aplikaci Microsoft Dynamics 365 Commerce Store Commerce.

- Při odstraňování problémů s výkonem pro Store Commerce se vyhněte vzdálenému přihlašování. Místo toho se přihlaste přímo do zařízení Store Commerce.
- Pokud je aplikace Store Commerce pomalá, zkontrolujte využití procesoru, paměti, sítě a disku pomocí Správce úloh nebo Sledování prostředků. Ujistěte se, že nedochází k významné spotřebě zdrojů zařízení.
- Zkontrolujte stav připojení k síti a potvrďte, že požadavky/odpovědi HTTPS netrvají déle než několik sekund.
