1. <span data-ttu-id="a151d-112">Vyberte obrázek pro **Režim úprav**.</span><span class="sxs-lookup"><span data-stu-id="a151d-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="a151d-113">V **Režimu úprav** vyberte **Upravit zobrazení podle modulu**.</span><span class="sxs-lookup"><span data-stu-id="a151d-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>

1. <span data-ttu-id="a151d-116">Z rozevírací nabídky **mmístění** vyberte umístění obrázku.</span><span class="sxs-lookup"><span data-stu-id="a151d-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="a151d-117">Z rozevírací nabídky **Zobrazení** vyberte velikost zobrazení.</span><span class="sxs-lookup"><span data-stu-id="a151d-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="a151d-118">Obrázek je překrytý oblastí, která představuje oblast oříznutí.</span><span class="sxs-lookup"><span data-stu-id="a151d-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="a151d-119">Podle potřeby přesuňte nebo změňte velikost oblasti oříznutí.</span><span class="sxs-lookup"><span data-stu-id="a151d-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="a151d-120">Poměr stran bude zachován automaticky.</span><span class="sxs-lookup"><span data-stu-id="a151d-120">The aspect ratio will be maintained automatically.</span></span>
