## <a name="crop-an-image"></a><span data-ttu-id="1b591-106">Oříznout obrázek</span><span class="sxs-lookup"><span data-stu-id="1b591-106">Crop an image</span></span>

<span data-ttu-id="1b591-107">Chcete-li oříznout obrázek v rámci konfigurátoru webu, postupujte podle následujících kroků.</span><span class="sxs-lookup"><span data-stu-id="1b591-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="1b591-108">V levém navigačním podokně konfigurátoru webu Commerce vyberte položku **Knihovna médií**.</span><span class="sxs-lookup"><span data-stu-id="1b591-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="1b591-109">V hlavním okně vyberte obrázek, který chcete upravit.</span><span class="sxs-lookup"><span data-stu-id="1b591-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="1b591-110">Na příkazovém řádku vyberte možnost **Upravit**.</span><span class="sxs-lookup"><span data-stu-id="1b591-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="1b591-111">Vyberte obrázek pro **Režim úprav**.</span><span class="sxs-lookup"><span data-stu-id="1b591-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="1b591-112">V **Režimu úprav** vyberte **Upravit zobrazení podle modulu**.</span><span class="sxs-lookup"><span data-stu-id="1b591-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="1b591-113">Z rozevírací nabídky **Modul** vyberte typ modulu.</span><span class="sxs-lookup"><span data-stu-id="1b591-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="1b591-114">Z rozevírací nabídky **Typ zobrazení** vyberte typ zobrazení.</span><span class="sxs-lookup"><span data-stu-id="1b591-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="1b591-115">Z rozevírací nabídky **mmístění** vyberte umístění obrázku.</span><span class="sxs-lookup"><span data-stu-id="1b591-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="1b591-116">Z rozevírací nabídky **Zobrazení** vyberte velikost zobrazení.</span><span class="sxs-lookup"><span data-stu-id="1b591-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="1b591-117">Obrázek je překrytý oblastí, která představuje oblast oříznutí.</span><span class="sxs-lookup"><span data-stu-id="1b591-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="1b591-118">Podle potřeby přesuňte nebo změňte velikost oblasti oříznutí.</span><span class="sxs-lookup"><span data-stu-id="1b591-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="1b591-119">Poměr stran bude zachován automaticky.</span><span class="sxs-lookup"><span data-stu-id="1b591-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="1b591-120">Až skončíte, vyberte v panelu příkazů **Uložit** a pak vyberte **Dokončit úpravy**.</span><span class="sxs-lookup"><span data-stu-id="1b591-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="1b591-121">Po dokončení vlastního oříznutí se úpravy obrazu projeví téměř okamžitě.</span><span class="sxs-lookup"><span data-stu-id="1b591-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
