## <a name="overview"></a><span data-ttu-id="bd9bf-105">Přehled</span><span class="sxs-lookup"><span data-stu-id="bd9bf-105">Overview</span></span>

<span data-ttu-id="bd9bf-106">Ikona oblíbené položky je malý grafický soubor, který se mimo jiná umsítění zobrazuje na kartě webového prohlížeče, v adresním řádku, v historii procházení a v záložkách nebo oblíbených položkách.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="bd9bf-107">Doporučujeme přidat ikonu oblíbené položky na svůj web, protože reprezentuje a posiluje vaši značku a pomáhá rozlišit váš web od jiných webů, které vaši zákazníci navštěvují.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="bd9bf-108">Ačkoli lze na web přidat více ikon oblíbené položky různých velikostí a typů souborů, tohle téma ukazuje, jak přidat jednu ikonu oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="bd9bf-109">Stejný proces a umístění se však používají k přidání dalších ikon oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="bd9bf-110">Odeslání ikony oblíbené položky do kolekce prostředků webu</span><span class="sxs-lookup"><span data-stu-id="bd9bf-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="bd9bf-111">Chcete-li odeslat ikonu oblíbené položky do kolekce prostředků vašeho webu, postupujte následovně.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="bd9bf-112">V levém navigačním podokně vyberte položku **Knihovna médií**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="bd9bf-113">Na panelu příkazů vyberte možnost **Odeslat \> Odeslání mediálních položek**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="bd9bf-114">V okně Průzkumník souborů přejděte na obrazový soubor ikony oblíbené položky, který chcete nahrát, vyberte jej a poté vyberte **Otevřít**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>

1. <span data-ttu-id="bd9bf-116">Chcete-li publikovat obrázky ihned po odeslání, zaškrtněte políčko **Publikovat mediální položky po odeslání**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="bd9bf-117">Pokud nezaškrtnete políčko **Publikovat prostředky po odeslání**, je nutné vrátit se na stránku **Prostředky** a ručně publikovat ikonu oblíbené položky později.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="bd9bf-118">Vyberte **OK**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-118">Select **OK**.</span></span>
1. <span data-ttu-id="bd9bf-119">V podokně vlastností vpravo zkopírujte veřejnou adresu URL ikony oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="bd9bf-120">Tuto adresu URL budete později používat.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-120">You will use this URL later.</span></span>

## <a name="create-a-page-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="bd9bf-124">Vytvořte fragment stránky, který obsahuje metaznačku pro vaši ikonu oblíbené položky</span><span class="sxs-lookup"><span data-stu-id="bd9bf-124">Create a page fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="bd9bf-125">Chcete-li vytvořit fragment stránky, který obsahuje metaznačku pro ikonu oblíbené položky, postupujte následovně.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-125">To create a page fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="bd9bf-126">Přejděte na **Fragmenty stránky** a pak vyberte **Nový**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-126">Go to **Page fragments**, and select **New**.</span></span>
1. <span data-ttu-id="bd9bf-127">V dialogovém okně **Nový fragment stránky** vyberte jako modul, na němž je založen fragment stránky **Metaznačky**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-127">In the **New Page Fragment** dialog box, select **Metatags** as the module that the page fragment is based on.</span></span>
1. <span data-ttu-id="bd9bf-128">Zadejte název fragmentu stránky a poté vyberte **OK**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-128">Enter a name for the page fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="bd9bf-129">Ve stromu hierarchie fragmentů vyberte podřízenou položku **Výchozí metaznačky**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="bd9bf-130">V pravém podokně pod **Metaznačky** vyberte **Přidat** a zadejte řetězec HTML, který jste pro oblíbenou položku vytvořili dříve.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span>

1. <span data-ttu-id="bd9bf-134">Přejděte na **Šablony** a otevřete šablonu pro stránky, na které chcete přidat ikonu oblíbené položky. Potom vyberte **Upravit**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="bd9bf-135">Ve stromu hierarchie šablon vyberte tlačítko se třemi tečkami (**...**) napravo od kontejneru **Hlavička HTML** a vyberte **Přidat fragment stránky**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add page fragment**.</span></span>
1. <span data-ttu-id="bd9bf-136">V dialogovém okně **Vybrat fragment stránky** vyberte fragment stránky metaznačky, který jste vytvořili předtím, a pak vyberte tlačítko **OK**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-136">In the **Select Page Fragment** dialog box, select the metatag page fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="bd9bf-137">Chcete-li publikovat šablonu, vyberte možnost **Dokončit úpravy** a volbou **Publikovat**.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="bd9bf-138">Pokud váš web používá více než jednu šablonu, musíte do každé z nich přidat fragment stránky s metaznačkami.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-138">If your site uses more than one template, you must add the metatags page fragment to all of them.</span></span>

<span data-ttu-id="bd9bf-139">Při náhledu stránek založených na šabloně, do které jste přidali fragment stránky metaznaček, byste nyní měli na kartě prohlížeče vidět ikonu oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="bd9bf-139">When you preview pages that are based on the template that you added the metatags page fragment to, you should now see the favicon on the browser tab.</span></span>
