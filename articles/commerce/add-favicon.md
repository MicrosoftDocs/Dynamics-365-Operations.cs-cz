## <a name="overview"></a><span data-ttu-id="15a73-105">Přehled</span><span class="sxs-lookup"><span data-stu-id="15a73-105">Overview</span></span>

<span data-ttu-id="15a73-106">Ikona oblíbené položky je malý grafický soubor, který se mimo jiná umsítění zobrazuje na kartě webového prohlížeče, v adresním řádku, v historii procházení a v záložkách nebo oblíbených položkách.</span><span class="sxs-lookup"><span data-stu-id="15a73-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="15a73-107">Doporučujeme přidat ikonu oblíbené položky na svůj web, protože reprezentuje a posiluje vaši značku a pomáhá rozlišit váš web od jiných webů, které vaši zákazníci navštěvují.</span><span class="sxs-lookup"><span data-stu-id="15a73-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="15a73-108">Ačkoli lze na web přidat více ikon oblíbené položky různých velikostí a typů souborů, tohle téma ukazuje, jak přidat jednu ikonu oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="15a73-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="15a73-109">Stejný proces a umístění se však používají k přidání dalších ikon oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="15a73-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="15a73-110">Odeslání ikony oblíbené položky do kolekce prostředků webu</span><span class="sxs-lookup"><span data-stu-id="15a73-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="15a73-111">Chcete-li odeslat ikonu oblíbené položky do kolekce prostředků vašeho webu, postupujte následovně.</span><span class="sxs-lookup"><span data-stu-id="15a73-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="15a73-114">V okně Průzkumník souborů přejděte na obrazový soubor ikony oblíbené položky, který chcete nahrát, vyberte jej a poté vyberte **Otevřít**.</span><span class="sxs-lookup"><span data-stu-id="15a73-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>

1. <span data-ttu-id="15a73-116">Chcete-li publikovat obrázky ihned po odeslání, zaškrtněte políčko **Publikovat mediální položky po odeslání**.</span><span class="sxs-lookup"><span data-stu-id="15a73-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="15a73-117">Pokud nezaškrtnete políčko **Publikovat prostředky po odeslání**, je nutné vrátit se na stránku **Prostředky** a ručně publikovat ikonu oblíbené položky později.</span><span class="sxs-lookup"><span data-stu-id="15a73-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="15a73-118">Vyberte **OK**.</span><span class="sxs-lookup"><span data-stu-id="15a73-118">Select **OK**.</span></span>

1. <span data-ttu-id="15a73-128">Zadejte název fragmentu a poté vyberte **OK**.</span><span class="sxs-lookup"><span data-stu-id="15a73-128">Enter a name for the fragment, and then select **OK**.</span></span>

1. <span data-ttu-id="15a73-134">Přejděte na **Šablony** a otevřete šablonu pro stránky, na které chcete přidat ikonu oblíbené položky. Potom vyberte **Upravit**.</span><span class="sxs-lookup"><span data-stu-id="15a73-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="15a73-135">Ve stromu hierarchie šablon vyberte tlačítko se třemi tečkami (**...**) napravo od kontejneru **Hlavička HTML** a vyberte **Přidat fragment**.</span><span class="sxs-lookup"><span data-stu-id="15a73-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
1. <span data-ttu-id="15a73-136">V dialogovém okně **Vybrat fragment** vyberte fragment metaznačky, který jste vytvořili předtím, a pak vyberte tlačítko **OK**.</span><span class="sxs-lookup"><span data-stu-id="15a73-136">In the **Select fragment** dialog box, select the metatag fragment that you created earlier, and then select **OK**.</span></span>

> [!NOTE]
> <span data-ttu-id="15a73-138">Pokud váš web používá více než jednu šablonu, musíte do každé z nich přidat fragment s metaznačkami.</span><span class="sxs-lookup"><span data-stu-id="15a73-138">If your site uses more than one template, you must add the metatags fragment to all of them.</span></span>

<span data-ttu-id="15a73-139">Při náhledu stránek založených na šabloně, do které jste přidali fragment metaznaček, byste nyní měli na kartě prohlížeče vidět ikonu oblíbené položky.</span><span class="sxs-lookup"><span data-stu-id="15a73-139">When you preview pages that are based on the template that you added the metatags fragment to, you should now see the favicon on the browser tab.</span></span>
