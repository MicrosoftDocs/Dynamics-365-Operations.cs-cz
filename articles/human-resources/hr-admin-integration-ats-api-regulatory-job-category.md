# <a name="regulatory-job-category"></a>Regulační kategorie pracovních míst

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Toto téma popisuje stav nastavený pro regulační kategorie pracovních míst Dynamics 365 Human Resources.

Fyzický název: mshr_hcmregulatoryjobcategory

Tento výčet určuje sadu možností pro hodnoty sady možností regulační kategorie pracovních míst. To je k dispozici v existující sadě možností mshr_hcmregulatoryjobcategory.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Není | Žádný. |
| 200000001 | Vedení | Vedení/management. |
| 200000002 | Manažer | Manažeři na nižší a střední úrovně. |
| 200000003 | Profesionální | Profesionálové. |
| 200000004 | Technik | Technici. |
| 200000005 | SalesWorker | Pracovníci prodeje. |
| 200000006 | Administrativní | Pracovníci administrativní podpory. |
| 200000007 | CraftWorker | Řemeslníci. |
| 200000008 | Pracovník | Pracovníci. |
| 200000009 | Dělník | Dělníci/výpomoc. |
| 200000010 | ServiceWorker | Servisní pracovníci. |
