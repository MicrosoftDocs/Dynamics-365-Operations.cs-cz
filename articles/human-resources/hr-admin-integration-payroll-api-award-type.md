# <a name="award-type"></a>Typ odměny


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Toto téma popisuje sadu možností typu Odměna pro Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Sada možností: Typ odměny

Fyzický název: mshr_HrmCompVarAwardEmplType

Tento výčet poskytuje sadu možností pro typ odměry variabilní kompenzace ve virtuálních entitách Human Resources.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Obecná odměna | Obecná odměna. |
| 200000001 | Fixní paušální platba | Fixní paušální platba. |
