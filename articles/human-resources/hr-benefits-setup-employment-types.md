# <a name="set-up-employment-types"></a>Nastavení typů zaměstnání

[!include [banner](includes/preview-feature.md)]

Nastavte typy zaměstnání v aplikaci Dynamics 365 Human Resources tak, aby vyhovovaly vašim pracovníkům.

1. V pracovním prostoru **Správa výhod** vyberte v části **Nastavení** možnost **typy zaměstnání**.

2. Zvolte **Nové**.

3. Zadejte hodnoty pro zbývající pole:

   | Pole | Popis |
   | --- | --- |
   | Typ zaměstnání | Jedinečná identifikace typu zaměstnání. |
   | Popis | Popis typu zaměstnání. |

4. Zvolte **Uložit**.
