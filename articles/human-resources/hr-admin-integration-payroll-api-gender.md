# <a name="payroll---gender-option-set"></a>Mzda – sada možností pohlaví


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Toto téma popisuje sadu možností typu Pohlaví pro Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Sada možností: možnost pohlaví

Tento výčet poskytuje sadu možností týkajících se pohlaví zaměstnance. Jedná se o sadu možností mshr_hcmpersongender.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Není | Pohlaví nebylo zadáno. |
| 200000001 | Muž | Muž. |
| 200000002 | Žena | Žena. |
| 200000003 | Nespecifické | Výběr nespecifického pohlaví. |
