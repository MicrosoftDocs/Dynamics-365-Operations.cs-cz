# <a name="gender"></a>Rod

Toto téma popisuje sadu možností typu Pohlaví pro Dynamics 365 Human Resources.

Fyzický název: mshr_hcmpersongender

Tento výčet poskytuje sadu možností týkajících se pohlaví kandidáta. Jedná se o sadu možností mshr_hcmpersongender.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Není | Pohlaví nebylo zadáno. |
| 200000001 | Muž | Muž. |
| 200000002 | Žena | Žena. |
| 200000003 | Nespecifické | Výběr nespecifického pohlaví. |
