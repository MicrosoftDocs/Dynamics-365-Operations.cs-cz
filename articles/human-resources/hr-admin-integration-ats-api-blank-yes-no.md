Tento výčet poskytuje sadu možností pro vlastnosti ano/ne, která povoluje i mezery.

| Hodnota | Štítek | popis |

| 200000000 | Bianko | Nebyla vybrána žádná hodnota. |
| 200000001 | Ano | Ano. |
| 200000002 | Ne | Ne |
