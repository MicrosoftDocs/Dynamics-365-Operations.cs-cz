Tento článek popisuje sadu možností ohledně rodinného stavu pro Dynamics 365 Human Resources.

Tento výčet poskytuje sadu možností pro rodinný stav osobních záznamů.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Není | Nebyla vybrána žádná hodnota.
| 200000001 | Jedno | Svobodný/svobodná. |
| 200000002 | Ženatý/vdaná | Ženatý/vdaná. |
| 200000003 | Rozvedený/rozvedená | Rozvedený/rozvedená. |
| 200000004 | Vdovec/vdova | Vdovství. |
