[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="8134b-104">Pokud máte dobrovolníky nebo jiné zaměstnance, možná budete chtít změnit název pracovního prostoru **Zaměstnanecká samoobsluha**.</span><span class="sxs-lookup"><span data-stu-id="8134b-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="8134b-105">Tento pracovní prostor můžete změnit na **Samoobsluha**.</span><span class="sxs-lookup"><span data-stu-id="8134b-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="8134b-106">Změna názvu pracovního prostoru **Zaměstnanecká samoobsluha** také mění položku nabídky, kterou interně používá Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="8134b-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="8134b-107">Pokud jste dříve použili přizpůsobení zabezpečení položky nabídky **HcmEmployeeSelfServiceWorkspace**, doporučujeme použít stejné změny na **HcmSelfServiceWorkspace** a udržovat paritu.</span><span class="sxs-lookup"><span data-stu-id="8134b-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="8134b-108">V Human Resources vyberte možnost **Správa zaměstnanců**, vyberte **Odkazy** a poté vyberte **Parametry lidských zdrojů**.</span><span class="sxs-lookup"><span data-stu-id="8134b-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="8134b-109">Vyberte kartu **Zaměstnanecká samoobsluha**.</span><span class="sxs-lookup"><span data-stu-id="8134b-109">Select the **Employee self-service** tab.</span></span>

   ![Změna názvu samoobslužného pracovního prostoru zaměstnance na Samoobsluhu](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="8134b-112">Zvolte **Uložit**.</span><span class="sxs-lookup"><span data-stu-id="8134b-112">Select **Save**.</span></span>
