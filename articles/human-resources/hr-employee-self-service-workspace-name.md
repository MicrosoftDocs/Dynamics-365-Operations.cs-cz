> [!NOTE]
> <span data-ttu-id="1c7b4-106">Změna názvu pracovního prostoru **Zaměstnanecká samoobsluha** také mění položku nabídky, kterou interně používá Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="1c7b4-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="1c7b4-107">Pokud jste dříve použili přizpůsobení zabezpečení položky nabídky **HcmEmployeeSelfServiceWorkspace**, doporučujeme použít stejné změny na **HcmSelfServiceWorkspace** a udržovat paritu.</span><span class="sxs-lookup"><span data-stu-id="1c7b4-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>
