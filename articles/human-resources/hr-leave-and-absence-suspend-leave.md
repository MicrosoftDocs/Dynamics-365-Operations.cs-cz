# <a name="suspend-leave"></a><span data-ttu-id="a978c-103">Přerušit pracovní volno</span><span class="sxs-lookup"><span data-stu-id="a978c-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="a978c-104">Můžete pozastavit pracovní volno pro zaměstnance, abyste zastavili zpracování časového rozlišení pracovního volna u vybraných typů pracovního volna.</span><span class="sxs-lookup"><span data-stu-id="a978c-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="a978c-105">Pozastavit pracovní volno a absenci pro zaměstnance</span><span class="sxs-lookup"><span data-stu-id="a978c-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="a978c-106">V záznamu zaměstnance vyberte možnost **Pracovní volno**.</span><span class="sxs-lookup"><span data-stu-id="a978c-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="a978c-107">Vyberte **Přerušit pracovní volno**.</span><span class="sxs-lookup"><span data-stu-id="a978c-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="a978c-108">Zvolte **Nové**.</span><span class="sxs-lookup"><span data-stu-id="a978c-108">Select **New**.</span></span>

4. <span data-ttu-id="a978c-109">V dialogovém okně **Přerušit časové rozlišení pracovního volna** vyberte **Typ pracovního volna** spolu s **Počátečním datem** a **Koncovým datem** přerušení.</span><span class="sxs-lookup"><span data-stu-id="a978c-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

<span data-ttu-id="a978c-111">Pokud jsou časové rozlišení zpracovávány i v době, kdy je pracovní volno zaměstnance přerušeno, nebude pro typy přerušeného volna provedeno žádné časové rozlišení.</span><span class="sxs-lookup"><span data-stu-id="a978c-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
