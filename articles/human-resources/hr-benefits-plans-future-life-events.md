# <a name="configure-future-life-events"></a>Konfigurace budoucích životních událostí

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]
[!include [banner](../includes/preview-banner.md)]

V aplikaci Dynamics 365 Human Resources můžete naplánovat budoucí životní události.

1. V pracovním prostoru **Správa výhod** vyberte v části **Nastavení** možnost **budoucí životní události**.

2. Zvolte **Nové**.

3. Zadejte hodnoty pro zbývající pole:

   | Pole | Popis |
   | --- | --- |
   | Datum životní události | Systém zpracuje všechny události během období registrace, ke kterému dojde do tohoto data. |
   | Životní událost zaznamenána | Datum a čas zaznamenání životní události. |
   | Typ protokolu | Zobrazí, zda je akce jedna z následujících možností:</br></br>- **Aktualizovat** – Změna stávajícího záznamu, který sleduje životní události</br></br>- **Vložit** – vytvoření nového záznamu životní události |
   | ID typu životních událostí | Jedinečný identifikátor typu životní události. |
   | Typy životní události | Katalyzátor pro aktualizaci přihlášení zaměstnaneckých výhod. Další informace naleznete v části spouštěčů životní události. |
   | Stav | Určení, zda byla životní událost zpracována nebo ne. |
   | Řádek | Číslo řádku budoucí životní události. |

4. Zvolte možnost **Uložit**. 

Můžete odstranit budoucí životní události. Pokud je vymazána zpracovaná budoucí životní událost, vymaže se i budoucí záznam.
