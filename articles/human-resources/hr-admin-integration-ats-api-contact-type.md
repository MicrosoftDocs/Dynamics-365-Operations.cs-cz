# <a name="contact-type"></a>Typ kontaktu

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Toto téma popisuje sadu možností typu Kontakt pro Dynamics 365 Human Resources.

Fyzický název: mshr_logisticselectronicaddressmethodtype

Tento výčet poskytuje sadu možností pro hodnoty typů kontaktu na kandidáty. 

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Není | Není vybrán žádný typ. |
| 200000001 | Telefon | Telefonní číslo |
| 200000002 | E-mailová adresa | E-mailová adresa. |
| 200000003 | Adresa URL | Adresa URL webu. |
| 200000004 | Telex | Číslo telexu. |
| 200000005 | Fax | Číslo faxu. |
| 200000006 | Facebook | Účet Facebook. Identifikován podle ID uživatele. |
| 200000007 | Twitter | Účet Twitter. Identifikováno pomocí @uživatelské_jméno. |
| 200000008 | LinkedIn | Účet LinkedIn. Identifikován podle uživatelského jména. |
