# <a name="skill-level-type"></a>Typ úrovně dovedností

Toto téma popisuje stav nastavený pro úroveň dovednosti Dynamics 365 Human Resources.

Fyzický název: mshr_hrmskillleveltype

Tento výčet poskytuje sadu možností hodnot, které kategorizují úroveň přiřazenou dovednosti kandidáta.

| Hodnota | Štítek | popis |
| --- | --- | --- |
| 200000000 | Skutečný | Skutečná úroveň dovedností. |
| 200000001 | Cíl | Cílená hodnota pro úroveň dovedností. |
