# <a name="ScreenDestinationType">Cíl obrazovky</a>

[!include [banner](../includes/banner.md)]

Můžete konfigurovat cíl obrazovky pro každou komponentu složky nebo souboru formátu elektronického výkaznictví, který je nakonfigurován pro generování odchozích dokumentů. Na základě nastavení cíle se vygenerovaný dokument otevře pro náhled na samostatné kartě prohlížeče.

Nastavíte-li **Povoleno** na **Ano**, vytvoří se náhled výstupu. Některé typy souborů, například XML, TXT nebo PDF můžete zobrazit přímo v okně prohlížeče. U jiných typů souborů, jako jsou soubory aplikací Microsoft Excel nebo Word, se používá služba Microsoft Office Online.
