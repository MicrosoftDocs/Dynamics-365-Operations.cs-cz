# <a name="power-bi-destination"></a>Cílové místo Power BI

[!include [banner](../includes/banner.md)]

Můžete konfigurovat cíl Microsoft Power BI pro každou složku nebo komponent souboru formátu elektronického výkaznictví, který je nakonfigurován pro generování odchozích dokumentů. V závislosti na nastavení cíle je vygenerovaný dokument uložen v dříve konfigurované složce SharePoint.

Nastavte **Povoleno** na **Ano** pro použití vaší konfigurace elektronického výkaznictví k uspořádání přenosu dat z instance Dynamics 365 Finance do služeb Microsoft Power BI. Převedené soubory se ukládají na instanci serveru Microsoft SharePoint, který musí být konfigurován pro tento účel. Další informace získáte v tématu [Konfigurace elektronického výkaznictví (ER) pro doplňování dat do Power BI](general-electronic-reporting-report-configuration-get-data-powerbi.md).
