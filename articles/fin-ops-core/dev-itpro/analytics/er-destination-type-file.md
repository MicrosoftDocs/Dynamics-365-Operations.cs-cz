# <a name="file-destination"></a>Místo určení souboru

[!include [banner](../includes/banner.md)]

Můžete konfigurovat cíl souboru pro každou komponentu SLOŽKA nebo SOUBOR formátu elektronického výkaznictví, který je nakonfigurován pro generování odchozích dokumentů. Na základě nastavení cíle je vygenerovaný dokument dostupný ke stažení z prohlížeče.

Když na stránce **Nastavení cíle** nastavíte **Povoleno** na **Ano**, otevře se po dokončení konfigurace dialogové okno pro otevření nebo uložení.
