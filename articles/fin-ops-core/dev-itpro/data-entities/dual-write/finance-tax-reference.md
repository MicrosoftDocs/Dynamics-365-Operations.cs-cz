# <a name="access-to-finance-and-tax-reference-data"></a>Přístup k referenčním datům financování a daně

[!include [banner](../../includes/banner.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Každý podnik pracuje se základní sadou finančních dat, jako je fiskální kalendářní rok, měna, v níž probíhají transakce, účty, na které přicházejí peníze pro provoz podniku, nebo ze kterého peníze odcházejí, daňové sazby a úhrady. Tato data se nacházejí v aplikacích Finance and Operations. Ty jsou však k dispozici pro Dataverse, takže aplikace Customer Engagement mohou mít jediný zdroj pro data financování a daně. Tímto způsobem jsou data jednotná napříč obchodním ekosystémem.

Data financování a daně jsou integrována pomocí následujících mapování:

+ [Integrovaná hlavní kniha](ledger-mapping.md)
+ [Integrovaná hlavní data daní](tax-mapping.md)
