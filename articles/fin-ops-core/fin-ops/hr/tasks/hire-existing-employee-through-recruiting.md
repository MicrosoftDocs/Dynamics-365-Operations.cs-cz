[!include [banner](../../includes/banner.md)]

<span data-ttu-id="05116-104">Otevřené pozice mohou někdy zaplnit kandidáti, kteří jsou již zaměstnanci ve vaší organizaci.</span><span class="sxs-lookup"><span data-stu-id="05116-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="05116-105">Tento postup vás provede kroky zařazení existujícího zaměstnance prostřednictvím procesu náboru.</span><span class="sxs-lookup"><span data-stu-id="05116-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="05116-106">V tomto postupu je již nastaven náborový projekt a existující zaměstnanec již odeslal přihlášku týkající se daného náborového projektu.</span><span class="sxs-lookup"><span data-stu-id="05116-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="05116-107">K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="05116-107">The demo data company used to create this procedure is USMF.</span></span>

3. <span data-ttu-id="05116-111">Klikněte na Stav uzamčení aplikace.</span><span class="sxs-lookup"><span data-stu-id="05116-111">Click Application status.</span></span>
    * <span data-ttu-id="05116-112">Stav přihlášky označuje, kde se přihláška nachází v procesu náboru.</span><span class="sxs-lookup"><span data-stu-id="05116-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="05116-113">Každý z těchto kroků je volitelný.</span><span class="sxs-lookup"><span data-stu-id="05116-113">Each of these steps is optional.</span></span> <span data-ttu-id="05116-114">Obvykle se přihláška přesunuje v rámci stavů v následujícím pořadí: přijato, potvrzeno a proběhl pohovor.</span><span class="sxs-lookup"><span data-stu-id="05116-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="05116-115">Po pohovoru bude učiněno rozhodnutí o nástupu.</span><span class="sxs-lookup"><span data-stu-id="05116-115">After the interview process, a hiring decision would be made.</span></span>

6. <span data-ttu-id="05116-118">V poli Počáteční datum nového přiřazení zadejte datum, kdy zaměstnanec nastoupí do práce na novou pozici.</span><span class="sxs-lookup"><span data-stu-id="05116-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>

    * <span data-ttu-id="05116-125">Nyní můžete také změnit kompenzaci.</span><span class="sxs-lookup"><span data-stu-id="05116-125">You may also change compensation at this time.</span></span> <span data-ttu-id="05116-126">Pokud nyní kompenzaci nepřiřadíte, můžete to změnit zobrazením formuláře pracovníka, výběrem karty Kompenzace a výběrem možnosti „Fixní plán“.</span><span class="sxs-lookup"><span data-stu-id="05116-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="05116-127">Po výběru možnosti „Změna pozice“ bude stav přihlášky aktualizován na hodnotu „Zaměstnán“.</span><span class="sxs-lookup"><span data-stu-id="05116-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
