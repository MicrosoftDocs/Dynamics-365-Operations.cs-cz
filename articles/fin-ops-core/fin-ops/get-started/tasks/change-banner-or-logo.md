# <a name="change-the-banner-or-logo"></a><span data-ttu-id="ac8a2-103">Změna nápisu nebo loga</span><span class="sxs-lookup"><span data-stu-id="ac8a2-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ac8a2-104">V následující proceduře je uveden postup, pomocí kterého mohou správci systému aktualizovat proužkovou reklamu nebo obrázek loga, které se zobrazí pro právnickou osobu.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="ac8a2-105">K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="ac8a2-106">Přejděte na **Navigační podokno > Moduly > Správa organizace > Organizace > Právnické osoby**.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="ac8a2-107">Na seznamu vlevo vyberte právnickou osobu, pro kterou chcete aktualizovat proužkovou reklamu nebo logo.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="ac8a2-108">Pokud je již vybrána, pokračujte dalším krokem.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="ac8a2-109">Rozbalte kartu **Obrázek řídicího panelu**.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="ac8a2-110">Vyberte **Změnit**.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-110">Select **Change**.</span></span>
    
    <span data-ttu-id="ac8a2-111">Ideální rozlišení pro obrázek s proužkovou reklamou je 1920 x 281 pixelů.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="ac8a2-112">Ideální šířka pro obrázek loga je 350 pixelů.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="ac8a2-113">Zvolte **Uložit**.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-113">Select **Save**.</span></span>
6. <span data-ttu-id="ac8a2-114">Přejděte na **Navigační podokno > Moduly > Běžné > Běžné > Výchozí řídicí panel**.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="ac8a2-115">Měli byste na řídícím panelu vidět novou proužkovou reklamu nebo obrázek loga.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="ac8a2-116">Může být nutné změnit vaší společnost pomocí pole pro výběr společnosti na tu, pro kterou jste odeslali proužkovou reklamu.</span><span class="sxs-lookup"><span data-stu-id="ac8a2-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
