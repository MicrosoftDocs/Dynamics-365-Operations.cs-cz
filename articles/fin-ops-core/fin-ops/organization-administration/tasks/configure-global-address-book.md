[!include [banner](../../includes/banner.md)]

<span data-ttu-id="14e3b-104">Tento postup slouží k nastavení výchozích hodnot a zásad zabezpečení pro globální adresář.</span><span class="sxs-lookup"><span data-stu-id="14e3b-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="14e3b-105">Tento úkol byl vytvořen pomocí ukázkových dat společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="14e3b-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="14e3b-106">Tento úkol je určen pro tým plánování a konfigurace.</span><span class="sxs-lookup"><span data-stu-id="14e3b-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="14e3b-107">V navigačním podokně přejděte na **Moduly > Správa organizace > Globální adresář > Parametry globálního adresáře**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="14e3b-108">V poli **Řada jmen** vyberte, jakým způsobem mají být zobrazeny názvy.</span><span class="sxs-lookup"><span data-stu-id="14e3b-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="14e3b-109">V okně **Odstranit strany bez rolí** vyberte, zda mají být odstraněny strany s touto rolí.</span><span class="sxs-lookup"><span data-stu-id="14e3b-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="14e3b-110">V poli **Použít kontrolu duplicit** vyberte, zda mají být zkontrolovány duplicitní záznamy.</span><span class="sxs-lookup"><span data-stu-id="14e3b-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="14e3b-111">V části **Zobrazit číslo DUNS v adresách** vyberte, zda se má v adresách zobrazit číslo DUNS.</span><span class="sxs-lookup"><span data-stu-id="14e3b-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="14e3b-112">V okně **Zkontrolovat jedinečné číslo DUNS** vyberte, zda chcete kontrolovat jedinečná čísla DUNS.</span><span class="sxs-lookup"><span data-stu-id="14e3b-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="14e3b-113">Vyberte možnost v poli **Strana**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="14e3b-114">Vyberte možnost v poli **Odběratel**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="14e3b-115">Vyberte možnost v poli **Dodavatel**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="14e3b-116">Vyberte možnost v poli **Potenciální zákazník**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="14e3b-117">Vyberte možnost v poli **Konkurent**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="14e3b-118">Klikněte na kartu **Zabezpečení soukromého** umístění.</span><span class="sxs-lookup"><span data-stu-id="14e3b-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="14e3b-119">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="14e3b-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="14e3b-120">Stiskněte klávesu Shift, chcete-li vybrat více rolí, které chcete přidat do podokna **Vybrané role**, a kliknutím na šipku vybrané role přidejte.</span><span class="sxs-lookup"><span data-stu-id="14e3b-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="14e3b-121">Klikněte na možnost **Uložit**.</span><span class="sxs-lookup"><span data-stu-id="14e3b-121">Click **Save**.</span></span>
