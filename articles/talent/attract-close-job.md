# <a name="close-a-job"></a><span data-ttu-id="1fd3b-103">Uzavření práce</span><span class="sxs-lookup"><span data-stu-id="1fd3b-103">Close a job</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="1fd3b-104">V Microsoft Dynamics 365 for Talent: Attract můžete pracovní nabídku uzavřít buď jako **Práca zrušena** nebo **Pozice obsazena**.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-104">In Microsoft Dynamics 365 for Talent: Attract, you can close a job as either **Job cancelled** or **Position filled**.</span></span> <span data-ttu-id="1fd3b-105">Práci lze kdykoli zrušit.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-105">You can cancel a job at any time.</span></span>

## <a name="cancel-a-job"></a><span data-ttu-id="1fd3b-107">Zrušit úlohu</span><span class="sxs-lookup"><span data-stu-id="1fd3b-107">Cancel a job</span></span>

<span data-ttu-id="1fd3b-108">Práci lze v Attract kdykoli zrušit.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-108">You can cancel a job in Attract at any time.</span></span>

1. <span data-ttu-id="1fd3b-109">Na kartě **Práce** v Attract vyberte nabídku, kterou chcete zrušit.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-109">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

   ![Zavřít úlohu v aplikaci Attract](./media/attract-close-job.png)

3. <span data-ttu-id="1fd3b-112">V dialogovém okně **Zavřít tuto úlohu** vyberte možnost **Práce zrušena**, přidejte všechny komentáře a poté vyberte možnost **Zavřít úlohu**.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-112">In the **Close this job** dialog box, select **Job cancelled**, add any comments, and then select **Close job**.</span></span>

   ![Uzavřít úlohu jako Pozice obsazena](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a><span data-ttu-id="1fd3b-114">Uzavřít úlohu jako Pozice obsazena</span><span class="sxs-lookup"><span data-stu-id="1fd3b-114">Close a job as Position filled</span></span>

<span data-ttu-id="1fd3b-115">Úlohu lze uzavřít jako **Pozice obsazena** poté, co alespoň jeden potenciální uchazeč dokončil fázi nabídky.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-115">You can close a job as **Position filled** after at least one prospect has completed the Offer stage.</span></span>

2. <span data-ttu-id="1fd3b-117">Vyberte **Zavřít práci**.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-117">Select **Close job**.</span></span>

3. <span data-ttu-id="1fd3b-119">V dialogovém okně **Zavřít tuto práci** vyberte možnost **Pozice obsazena**, vyberte uchazeče, který nabídku přijal, přidejte poznámky a poté vyberte možnost **Uzavřít práci**.</span><span class="sxs-lookup"><span data-stu-id="1fd3b-119">In the **Close this job** dialog box, select **Position filled**, select the candidate who accepted the offer, add any comments, and then select **Close job**.</span></span>
