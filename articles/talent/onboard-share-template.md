## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="ca950-105">Sdílení průvodce zaškolením s jinými přispěvateli</span><span class="sxs-lookup"><span data-stu-id="ca950-105">Share an onboarding guide with other contributors</span></span>

    ![[Sdílení průvodce zaškolením s ostatními přispěvateli] (./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="ca950-109">V části **Sdílet s**, pokud by příjemci měli mít možnost revidovat příručku, nastavte možnost **Mohou tyto osoby provádět úpravy?** na **Zapnuto**.</span><span class="sxs-lookup"><span data-stu-id="ca950-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="ca950-110">Pokud mají být příjemci upozorněni e-mailem, že jste s nimi nasdíleli průvodce, nastavte možnost **Odeslat e-mailové oznámení příjemcům** na **Zapnuto**.</span><span class="sxs-lookup"><span data-stu-id="ca950-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[Nastavení přístupu a předvoleb e-mailu pro průvodce zaškolením] (./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="ca950-112">Zadejte jméno nebo e-mailovou adresu a z rozevíracího seznamu vyberte požadovaného příjemce.</span><span class="sxs-lookup"><span data-stu-id="ca950-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="ca950-113">Na kartě **Spravovat přístup** můžete zkontrolovat nebo změnit možnosti, které příjemci mohou s průvodcem provádět.</span><span class="sxs-lookup"><span data-stu-id="ca950-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="ca950-114">Chcete-li změnit přístup příjemce, vyberte v rozevíracím seznamu vedle jména příjemce podle potřeby možnost **Může prohlížet**, **Může upravovat** nebo **Odebrat**.</span><span class="sxs-lookup"><span data-stu-id="ca950-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
