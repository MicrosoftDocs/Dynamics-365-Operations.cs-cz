[!include [banner](../../includes/banner.md)]

<span data-ttu-id="708e8-104">Hlavní kniha nákladového účetnictví představuje celkovou jednotku výkaznictví.</span><span class="sxs-lookup"><span data-stu-id="708e8-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="708e8-105">Je definována dimenzí nákladového prvku, statistickou dimenzí, fiskálním kalendářem a měnou.</span><span class="sxs-lookup"><span data-stu-id="708e8-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="708e8-106">Nerozlišuje princip právnických osob.</span><span class="sxs-lookup"><span data-stu-id="708e8-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="708e8-107">Právnická osoba a její data mohou být přidružena více hlavním knihám nákladového účetnictví.</span><span class="sxs-lookup"><span data-stu-id="708e8-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="708e8-108">Tento záznam používá v ukázkových datech společnost USP2.</span><span class="sxs-lookup"><span data-stu-id="708e8-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="708e8-109">Přejděte na Nákladové účetnictví > Nastavení hlavní knihy > Hlavní knihy nákladového účetnictví.</span><span class="sxs-lookup"><span data-stu-id="708e8-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>

3. <span data-ttu-id="708e8-111">Zadejte hodnotu do pole Název.</span><span class="sxs-lookup"><span data-stu-id="708e8-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="708e8-112">V poli Dimenze prvku nákladů zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="708e8-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="708e8-113">V poli Fiskální kalendář zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="708e8-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="708e8-114">V poli Měna účtu zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="708e8-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="708e8-115">V poli Typ směnného kurzu zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="708e8-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="708e8-116">V poli Statistická dimenze zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="708e8-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="708e8-117">Klikněte na položku Uložit.</span><span class="sxs-lookup"><span data-stu-id="708e8-117">Click Save.</span></span>
