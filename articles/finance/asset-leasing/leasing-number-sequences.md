[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="4ed76-105">Toto téma vysvětluje, jak vytvořit číselné řady pro ID leasingu.</span><span class="sxs-lookup"><span data-stu-id="4ed76-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="4ed76-106">Vysvětluje také, jak vytvořit jedinečná ID, která se používají v procesu přecenění indexu.</span><span class="sxs-lookup"><span data-stu-id="4ed76-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>
