[!include [banner](../includes/banner.md)]

<span data-ttu-id="60330-104">Toto téma vysvětluje, jak nastavit skupiny komponent srážkové daně, jako je **nájem** a **smluvní strana**, pro daňový typ se srážkou daně u zdroje (TDS).</span><span class="sxs-lookup"><span data-stu-id="60330-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="60330-105">Přejděte do nabídky **Daň \> Nastavení \> Srážková daň \> Skupiny komponent srážkové daně**.</span><span class="sxs-lookup"><span data-stu-id="60330-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="60330-106">[![Stránka Skupiny komponent srážkové daně](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="60330-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="60330-107">V poli **Typ daně** vyberte **TDS** a nastavte skupiny komponent srážkové daně pro typ daně TDS.</span><span class="sxs-lookup"><span data-stu-id="60330-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="60330-108">V podokně Akce vyberte možnost **Nový** a vytvořte řádek.</span><span class="sxs-lookup"><span data-stu-id="60330-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="60330-109">Do pole **Skupina komponent srážkové daně** zadejte název skupiny komponent TDS.</span><span class="sxs-lookup"><span data-stu-id="60330-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="60330-110">Zadejte popis do pole **Popis**.</span><span class="sxs-lookup"><span data-stu-id="60330-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="60330-111">Na pevné záložce **Obecné** v poli **Stav** vyberte rezidenční stav skupiny komponent TDS.</span><span class="sxs-lookup"><span data-stu-id="60330-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="60330-112">Možnosti jsou **Rezident** a **Nerezident**.</span><span class="sxs-lookup"><span data-stu-id="60330-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="60330-113">Do pole **Kód sekce** zadejte kód sekce, který se použije na skupinu komponent TDS.</span><span class="sxs-lookup"><span data-stu-id="60330-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="60330-114">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="60330-114">Close the page.</span></span>
