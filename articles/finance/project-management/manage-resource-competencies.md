## <a name="set-up-competencies-certificates"></a><span data-ttu-id="151fb-112">Nastavení kompetencí: certifikáty</span><span class="sxs-lookup"><span data-stu-id="151fb-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="151fb-113">Na stránce se seznamem **Pracovníci** vyberte řádek zaměstnance, pro kterého přidáváte informace o certifikátu.</span><span class="sxs-lookup"><span data-stu-id="151fb-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="151fb-117">Nastavení kompetencí: dovednosti</span><span class="sxs-lookup"><span data-stu-id="151fb-117">Set up competencies: Skills</span></span>

3. <span data-ttu-id="151fb-121">V poli **Dovednost** vyberte **Správa projektu**.</span><span class="sxs-lookup"><span data-stu-id="151fb-121">In the **Skill** field, select **Project management**.</span></span>

7. <span data-ttu-id="151fb-125">Zvolte **Uložit** a pak zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="151fb-125">Select **Save**, and then close the page.</span></span>
