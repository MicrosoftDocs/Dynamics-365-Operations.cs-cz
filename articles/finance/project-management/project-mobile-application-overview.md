# <a name="project-mobile-applications-overview"></a>Přehled mobilních aplikací projektu

[!include [banner](../includes/banner.md)]

Tři mobilní aplikace, které se vztahují k času projektu, jsou k dispozici pro Microsoft Dynamics 365 Project Timesheet, Časový záznam projektu a Moje časové rozvrhy/Časové rozvrhy pro mou kontrolu (optimalizované pro mobilní telefon).

## <a name="project-timesheet-mobile-app"></a>Mobilní aplikace Project Timesheet

Mobilní aplikace Dynamics 365 Project Timesheet umožňuje uživatelům odeslání a schválení časových rozvrhů projektů na jejich mobilních zařízeních. Tato mobilní aplikace se týká funkce časových rozvrhů v oblasti Řízení projektů a účetnictví v aplikaci Dynamics 365 Finance. Pomáhá zvýšit produktivitu a efektivitu uživatelů a zároveň pomáhá s včasným zadáváním a schvalováním časových rozvrhů projektu.

## <a name="project-time-entry-workspace"></a>Pracovní prostor časového záznamu projektu

Pracovní prostor časového záznamu projektu umožňuje uživatelům zadat a uložit čas proti projektu. Nepovoluje však uživatelům odesílat nebo schvalovat časové rozvrhy projektů. Slouží pouze pro zadávání času. Tento mobilní pracovní prostor je určen k použití spolu s mobilní aplikací Finance and Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Moje časové rozvrhy/Časové rozvrhy pro mou kontrolu

Moje časové rozvrhy (optimalizované pro mobilní telefon) a Časové rozvrhy pro mou kontrolu (optimalizované pro mobilní telefon) vytváří řešení, která umožní zaměstnancům odeslání a schválení časových rozvrhů projektu na mobilním zařízení. Pracovníci mohou přistupovat k funkci Finance prostřednictvím adresy URL aplikace. Toto řešení nevyžaduje mobilní aplikaci Finance and Operations.
