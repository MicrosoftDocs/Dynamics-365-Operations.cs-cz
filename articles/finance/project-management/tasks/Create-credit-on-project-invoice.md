# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="63a64-103">Vytvoření dobropisu na fakturách projektu</span><span class="sxs-lookup"><span data-stu-id="63a64-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="63a64-104">Přejděte na Řízení projektu a účetnictví > Projekty > Všechny projekty.</span><span class="sxs-lookup"><span data-stu-id="63a64-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="63a64-105">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="63a64-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="63a64-106">Vyberte ID projektu, který se používá pro dobropis.</span><span class="sxs-lookup"><span data-stu-id="63a64-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="63a64-107">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="63a64-107">In the list, click the link in the selected row.</span></span>

5. <span data-ttu-id="63a64-109">Klepněte na Deníky faktury.</span><span class="sxs-lookup"><span data-stu-id="63a64-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="63a64-110">Vyberte transakci pro dobropis.</span><span class="sxs-lookup"><span data-stu-id="63a64-110">Select the transaction for credit note.</span></span> <span data-ttu-id="63a64-111">Označte v seznamu vybraný řádek.</span><span class="sxs-lookup"><span data-stu-id="63a64-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="63a64-112">Klepněte na možnost Funkce.</span><span class="sxs-lookup"><span data-stu-id="63a64-112">Click Functions.</span></span> 
8. <span data-ttu-id="63a64-113">Klikněte na Výběr pro dobropis.</span><span class="sxs-lookup"><span data-stu-id="63a64-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="63a64-114">Zaškrtnutí políčka vyberte transakci pro dobropis.</span><span class="sxs-lookup"><span data-stu-id="63a64-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="63a64-115">Klikněte na tlačítko OK.</span><span class="sxs-lookup"><span data-stu-id="63a64-115">Click OK.</span></span> 

<span data-ttu-id="63a64-116">Po provedení tohoto kroku bude k dispozici dobropis pro návrh faktury.</span><span class="sxs-lookup"><span data-stu-id="63a64-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
