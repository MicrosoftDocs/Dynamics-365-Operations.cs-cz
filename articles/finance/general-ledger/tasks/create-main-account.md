[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="6cd2b-106">Přejděte do části **Navigační podokno > Moduly >Hlavní kniha > Účtová osnova > Účty > Hlavní účty.**</span><span class="sxs-lookup"><span data-stu-id="6cd2b-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>

3. <span data-ttu-id="6cd2b-108">Do pole **Hlavní účet** zadejte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="6cd2b-109">Zadejte hodnotu do pole **Název**.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="6cd2b-110">V poli **Typ hlavního účtu** vyberte typ, který nejlépe reprezentuje rozvahové účty a umístění ve finančních výkazech.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>

8. <span data-ttu-id="6cd2b-115">V poli **Výchozí měna** vyberte hodnotu ze seznamu měn.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="6cd2b-116">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="6cd2b-117">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="6cd2b-118">Přepněte rozšíření oddílu **Přepisy právnických osob**.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>

13. <span data-ttu-id="6cd2b-120">V seznamu vyberte položku Právnická položka.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="6cd2b-121">Klikněte na tlačítko **Přidat**.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-121">Click **Add**.</span></span>
15. <span data-ttu-id="6cd2b-122">Označte v seznamu vybraný řádek.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-122">In the list, mark the selected row.</span></span>

18. <span data-ttu-id="6cd2b-125">V poli **Typ směnného kurzu** otevřete vyhledávání klepnutím na tlačítko rozevíracího seznamu.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>

20. <span data-ttu-id="6cd2b-127">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="6cd2b-128">V poli **Typ převodu měny** vyberte metodu pro výpočet směnných kurzů pro účet.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="6cd2b-129">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="6cd2b-129">Close the page.</span></span>
