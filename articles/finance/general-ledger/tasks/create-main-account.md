# <a name="create-a-main-account"></a><span data-ttu-id="601e4-103">Vytvoření hlavního účtu</span><span class="sxs-lookup"><span data-stu-id="601e4-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="601e4-104">Tento průvodce úkoly váš provede přidáním hlavního účtu k existující účtové osnově.</span><span class="sxs-lookup"><span data-stu-id="601e4-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="601e4-105">Tento záznam používá ukázkovou společnost USMF.</span><span class="sxs-lookup"><span data-stu-id="601e4-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="601e4-106">Přejděte do části **Navigační podokno > Moduly >Hlavní kniha > Účtová osnova > Účty > Hlavní účty.**</span><span class="sxs-lookup"><span data-stu-id="601e4-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="601e4-107">Klepněte na možnost **Nový**.</span><span class="sxs-lookup"><span data-stu-id="601e4-107">Click **New**.</span></span>
3. <span data-ttu-id="601e4-108">Do pole **Hlavní účet** zadejte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="601e4-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="601e4-109">Zadejte hodnotu do pole **Název**.</span><span class="sxs-lookup"><span data-stu-id="601e4-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="601e4-110">V poli **Typ hlavního účtu** vyberte typ, který nejlépe reprezentuje rozvahové účty a umístění ve finančních výkazech.</span><span class="sxs-lookup"><span data-stu-id="601e4-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="601e4-111">Ze seznamu vyberte kategorii účtu, ke které patří hlavní účet.</span><span class="sxs-lookup"><span data-stu-id="601e4-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="601e4-112">Kategorie účtu je používána pro výchozí finanční výkazy a obsah řídicího panelu Power BI.</span><span class="sxs-lookup"><span data-stu-id="601e4-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="601e4-113">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="601e4-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="601e4-114">Změňte výchozí účet Má dáti nebo kreditní zůstatek.</span><span class="sxs-lookup"><span data-stu-id="601e4-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="601e4-115">V poli **Výchozí měna** vyberte hodnotu ze seznamu měn.</span><span class="sxs-lookup"><span data-stu-id="601e4-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="601e4-116">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="601e4-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="601e4-117">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="601e4-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="601e4-118">Přepněte rozšíření oddílu **Přepisy právnických osob**.</span><span class="sxs-lookup"><span data-stu-id="601e4-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="601e4-119">Chcete-li vybrat právnickou osobu, **klikněte na Přidat**.</span><span class="sxs-lookup"><span data-stu-id="601e4-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="601e4-120">V seznamu vyberte položku Právnická položka.</span><span class="sxs-lookup"><span data-stu-id="601e4-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="601e4-121">Klikněte na tlačítko **Přidat**.</span><span class="sxs-lookup"><span data-stu-id="601e4-121">Click **Add**.</span></span>
15. <span data-ttu-id="601e4-122">Označte v seznamu vybraný řádek.</span><span class="sxs-lookup"><span data-stu-id="601e4-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="601e4-123">Zaškrtněte nebo zrušte zaškrtnutí políčka **Pozastaveno**.</span><span class="sxs-lookup"><span data-stu-id="601e4-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="601e4-124">Rozbalte oddíl **Finanční výkaznictví**.</span><span class="sxs-lookup"><span data-stu-id="601e4-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="601e4-125">V poli **Typ směnného kurzu** otevřete vyhledávání klepnutím na tlačítko rozevíracího seznamu.</span><span class="sxs-lookup"><span data-stu-id="601e4-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="601e4-126">V seznamu vyberte **typ směnného kurzu pro účet**.</span><span class="sxs-lookup"><span data-stu-id="601e4-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="601e4-127">Klikněte na odkaz na vybraném řádku v seznamu.</span><span class="sxs-lookup"><span data-stu-id="601e4-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="601e4-128">V poli **Typ převodu měny** vyberte metodu pro výpočet směnných kurzů pro účet.</span><span class="sxs-lookup"><span data-stu-id="601e4-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="601e4-129">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="601e4-129">Close the page.</span></span>
