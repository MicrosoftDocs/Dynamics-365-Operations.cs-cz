# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="b404a-103">Zobrazení položek nebo transakcí deníku</span><span class="sxs-lookup"><span data-stu-id="b404a-103">View journal entries or transactions</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b404a-104">Tato procedura ukazuje, jak pomocí transakcí dokladu vyhledávat položky deníku nebo transakce.</span><span class="sxs-lookup"><span data-stu-id="b404a-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="b404a-105">Přejděte na **Navigační podokno > Moduly > Hlavní kniha > Dotazy a sestavy > Transakce dokladu**.</span><span class="sxs-lookup"><span data-stu-id="b404a-105">Go to **Navigation pane > Modules > General ledger > Inquiries and reports > Voucher transactions**.</span></span>
2. <span data-ttu-id="b404a-106">Vyberte pole, pro které chcete definovat kritéria filtru.</span><span class="sxs-lookup"><span data-stu-id="b404a-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="b404a-107">Zadejte kritéria filtru pro vybrané pole.</span><span class="sxs-lookup"><span data-stu-id="b404a-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="b404a-108">Můžete vyfiltrovat jednu hodnotu nebo rozsah hodnot.</span><span class="sxs-lookup"><span data-stu-id="b404a-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="b404a-109">Při definování rozsahu zkontrolujte, zda je použita správná syntaxe.</span><span class="sxs-lookup"><span data-stu-id="b404a-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="b404a-110">Hodnoty by měly být odděleny dvěma tečkami (..).</span><span class="sxs-lookup"><span data-stu-id="b404a-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="b404a-111">Pokud chcete přidat další tabulky k filtrování, klikněte na kartu **Spojení**.</span><span class="sxs-lookup"><span data-stu-id="b404a-111">Click the **Joins** tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="b404a-112">Ve stromové struktuře vyberte **Položku deníku Tables/General**.</span><span class="sxs-lookup"><span data-stu-id="b404a-112">In the tree, select **Tables/General journal entry**.</span></span>
6. <span data-ttu-id="b404a-113">Klikněte na **Přidat spojení tabulky**.</span><span class="sxs-lookup"><span data-stu-id="b404a-113">Click **Add table join**.</span></span>
7. <span data-ttu-id="b404a-114">Pokud se rozhodnete nepřidávat další tabulku, klikněte na tlačítko **Storno**.</span><span class="sxs-lookup"><span data-stu-id="b404a-114">Click **Cancel** if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="b404a-115">Klikněte na kartu **Rozsah**.</span><span class="sxs-lookup"><span data-stu-id="b404a-115">Click the **Range** tab.</span></span>
9. <span data-ttu-id="b404a-116">Dotaz spustíte klepnutím na tlačítko **OK**.</span><span class="sxs-lookup"><span data-stu-id="b404a-116">Click **OK** to run the query.</span></span>
10. <span data-ttu-id="b404a-117">V podokně akcí klikněte na možnost **Původ transakce**.</span><span class="sxs-lookup"><span data-stu-id="b404a-117">On the Action pane, click **Transaction origin**.</span></span> <span data-ttu-id="b404a-118">K vyhledání dalších informací o vybraném záznamu dokladu je možné použít různá tlačítka po celé mřížce.</span><span class="sxs-lookup"><span data-stu-id="b404a-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="b404a-119">Některá tlačítka nemusí být dostupná, v závislosti na typu transakce a vlastnostech transakce.</span><span class="sxs-lookup"><span data-stu-id="b404a-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
11. <span data-ttu-id="b404a-120">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="b404a-120">Close the page.</span></span>
12. <span data-ttu-id="b404a-121">V podokně akcí klikněte na možnost **Původní dokument**.</span><span class="sxs-lookup"><span data-stu-id="b404a-121">On the Action pane, Click **Original document**.</span></span>
13. <span data-ttu-id="b404a-122">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="b404a-122">Close the page.</span></span>
