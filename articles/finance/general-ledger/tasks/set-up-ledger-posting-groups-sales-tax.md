# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="0b13c-103">Nastavení účetních skupin pro DPH</span><span class="sxs-lookup"><span data-stu-id="0b13c-103">Set up Ledger posting groups for sales tax</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0b13c-104">DPH je vypočítáno a zaúčtováno do hlavních účtů, které jsou zadány v části Účetní skupiny.</span><span class="sxs-lookup"><span data-stu-id="0b13c-104">Sales tax is calculated and posted to main accounts that are specified in Ledger posting groups.</span></span> <span data-ttu-id="0b13c-105">Účetní skupiny jsou připojeny k jednotlivým kódům DPH.</span><span class="sxs-lookup"><span data-stu-id="0b13c-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="0b13c-106">Pro každý kód DPH můžete nastavit individuální účetní skupiny hlavní knihy pro DPH; můžete použít jednu účetní skupinu hlavní knihy pro všechny kódy DPH nebo můžete ke všem kódům DPH přiřadit několik účetních skupin hlavní knihy.</span><span class="sxs-lookup"><span data-stu-id="0b13c-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="0b13c-107">Tento záznam používá ukázkovou společnost DEMF.</span><span class="sxs-lookup"><span data-stu-id="0b13c-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="0b13c-108">Přejděte na **Navigační podokno > Moduly > Daň > Nastavení > DPH > Účetní skupiny**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="0b13c-109">Klepněte na možnost **Nový**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-109">Click **New**.</span></span>
3. <span data-ttu-id="0b13c-110">Zadejte hodnotu do pole **Skupina zaúčtování hl. knihy**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="0b13c-111">Zadejte hodnotu do pole **Popis**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="0b13c-112">V poli **DPH na výstupu** vyberte hlavní účet pro odchozí DPH, které jsou splatné finančnímu úřadu.</span><span class="sxs-lookup"><span data-stu-id="0b13c-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="0b13c-113">DPH se vybírají jménem finančního úřadu při prodeji zdanitelného zboží a služeb.</span><span class="sxs-lookup"><span data-stu-id="0b13c-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="0b13c-114">V poli **Pohledávka DPH** vyberte hlavní účet pro příchozí DPH, které jsou přijatelné od finančního úřadu.</span><span class="sxs-lookup"><span data-stu-id="0b13c-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="0b13c-115">Dodavatelé vybírají daně jménem finančního úřadu při nákupu zdanitelného zboží a služeb.</span><span class="sxs-lookup"><span data-stu-id="0b13c-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="0b13c-116">Toto pole není k dispozici v případě, že je vybrána možnost Použít daňové předpisy pro DPH na stránce **Parametry hlavní knihy**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="0b13c-117">DPH zaplacená dodavatelům je místo toho připsána na stejný účet, jako při nákupu.</span><span class="sxs-lookup"><span data-stu-id="0b13c-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="0b13c-118">V poli **Importní DPH – výdaj** vyberte hlavní účet pro zaúčtování odečitatelných importních daní, které nejsou dodavateli nárokovány ani vykazovány finančnímu úřadu jako součást stornovacího poplatku GST/HST pro EU.</span><span class="sxs-lookup"><span data-stu-id="0b13c-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="0b13c-119">Možnosti **Použít daň** musíte vybrat pro **kód DPH** ve **skupině DPH**, která je použitá v transakci.</span><span class="sxs-lookup"><span data-stu-id="0b13c-119">The **Use tax** option must be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="0b13c-120">Toto pole nebude k dispozici v případě, že je vybrána možnost **Použít daňové předpisy pro DPH** na stránce **Parametry hlavní knihy**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-120">This field won't be available if the **Apply sales tax taxation rules** option is selected on the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="0b13c-121">V poli **Splatná importní DPH** vyberte hlavní účet pro zaúčtování příchozích importních daní, které jsou splatné finančním úřadům.</span><span class="sxs-lookup"><span data-stu-id="0b13c-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="0b13c-122">Možnosti **Použít daň** musíte vybrat pro **kód DPH** ve **skupině DPH** pro zaúčtování hodnoty v části **Použít daň**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-122">The **Use tax** option must be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="0b13c-123">Pokud je vybrána možnost **Použít daňové předpisy pro DPH** na stránce **Parametry hlavní knihy**, protiúčet je zaúčtován na výdajový účet transakce.</span><span class="sxs-lookup"><span data-stu-id="0b13c-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="0b13c-124">V poli **Účet vyrovnání** vyberte hlavní účet na který bude zaúčtován čistý zůstatek účtů hlavní knihy podle pole **Splatná importní DPH** a **DPH na vstupu**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="0b13c-125">Zůstatek se vytvoří při provedení vyrovnání DPH a spuštění úlohy zúčtování.</span><span class="sxs-lookup"><span data-stu-id="0b13c-125">The balance will be created when the sales taxes settle and the posting job has been run.</span></span>  <span data-ttu-id="0b13c-126">Pokud je finančnímu úřadu pro období vyrovnání přidružen k účtu dodavatele, zůstatek se zaúčtuje namísto toho na účet dodavatele.</span><span class="sxs-lookup"><span data-stu-id="0b13c-126">If the taxing authority for the settlement period is associated with a vendor account, the balance will be posted to the vendor account instead.</span></span>
10. <span data-ttu-id="0b13c-127">V poli **Platební sleva dodavatele** vyberte hlavní účet pro zaúčtování platební slevy pro kódy DPH, které jsou přidružené k této skupině zaúčtování hlavní knihy.</span><span class="sxs-lookup"><span data-stu-id="0b13c-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="0b13c-128">Toto pole je nepovinné a pokud není zadán žádný účet, použije se hlavní účet ze seznamu **Kódy platebních slev**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="0b13c-129">Může být užitečné použití různých účtů pro každou **Skupinu zaúčtování hlavní knihy**, pokud používáte možnost Stornovat DPH u platební slevy pro skupiny DPH.</span><span class="sxs-lookup"><span data-stu-id="0b13c-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="0b13c-130">V poli **Platební sleva odběratele** vyberte hlavní účet pro zaúčtování platební slevy pro **Kódy DPH**, které jsou přidružené k této **Skupině zaúčtování hlavní knihy**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="0b13c-131">Toto pole je nepovinné a pokud není zadán žádný účet, použije se hlavní účet ze seznamu **Kódy platebních slev**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="0b13c-132">Může být užitečné použití různých účtů pro každou **Skupinu zaúčtování hlavní knihy**, pokud používáte možnost Stornovat DPH u platební slevy pro **skupiny DPH**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="0b13c-133">Klikněte na možnost **Uložit**.</span><span class="sxs-lookup"><span data-stu-id="0b13c-133">Click **Save**.</span></span>
