# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="3dea4-103">Vyřazení dlouhodobého majetku s použitím volné faktury</span><span class="sxs-lookup"><span data-stu-id="3dea4-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="3dea4-104">Tento postup popisuje vyřazení dlouhodobého majetku pomocí volné faktury.</span><span class="sxs-lookup"><span data-stu-id="3dea4-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="3dea4-105">Přejděte na Pohledávky > Faktury > Všechny volné faktury.</span><span class="sxs-lookup"><span data-stu-id="3dea4-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="3dea4-106">Klikněte na položku Nová.</span><span class="sxs-lookup"><span data-stu-id="3dea4-106">Click New.</span></span>
3. <span data-ttu-id="3dea4-107">V poli Účet odběratele zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="3dea4-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="3dea4-108">Ověřte výchozí datum faktury a podle potřeby ho upravte.</span><span class="sxs-lookup"><span data-stu-id="3dea4-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="3dea4-109">Ověřte zbývající výchozí pole záhlaví, jako je například Měna, a podle potřeby ho upravte.</span><span class="sxs-lookup"><span data-stu-id="3dea4-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="3dea4-110">Zadejte popis do řádku faktury.</span><span class="sxs-lookup"><span data-stu-id="3dea4-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="3dea4-111">Zadejte nebo zvolte Hlavní účet pro řádek faktury.</span><span class="sxs-lookup"><span data-stu-id="3dea4-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="3dea4-112">Ověřte výchozí skupinu DPH a skupinu DPH položky.</span><span class="sxs-lookup"><span data-stu-id="3dea4-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="3dea4-113">Zadejte jednotkovou cenu nebo množství prodeje dlouhodobého majetku.</span><span class="sxs-lookup"><span data-stu-id="3dea4-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="3dea4-114">Klikněte na Podrobnosti řádku.</span><span class="sxs-lookup"><span data-stu-id="3dea4-114">Click Line details.</span></span>  
11. <span data-ttu-id="3dea4-115">Vyberte číslo dlouhodobého majetku k prodeji.</span><span class="sxs-lookup"><span data-stu-id="3dea4-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="3dea4-116">Klikněte na položku Zaúčtovat.</span><span class="sxs-lookup"><span data-stu-id="3dea4-116">Click Post.</span></span>
