# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="e5b1e-103">Vyřazení dlouhodobého majetku s použitím volné faktury</span><span class="sxs-lookup"><span data-stu-id="e5b1e-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e5b1e-104">Toto téma vysvětluje vyřazení dlouhodobého majetku pomocí volné faktury.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="e5b1e-105">V podokně navigace přejděte na **Moduly > Pohledávky > Faktury > Všechny otevřené faktury**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="e5b1e-106">Zvolte **Nové**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-106">Select **New**.</span></span>
3. <span data-ttu-id="e5b1e-107">V poli **Účet odběratele** zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="e5b1e-108">Ověřte výchozí datum **faktury** a podle potřeby ho upravte.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="e5b1e-109">Ověřte zbývající výchozí pole záhlaví, jako je například **Měna**, a podle potřeby ho upravte.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="e5b1e-110">V části **Řádky faktury** ověřte pole **Popis** a **Hlavní účet**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="e5b1e-111">Ověřte výchozí pole **Skupina DPH** a **Skupina DPH položky**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="e5b1e-112">Zadejte **jednotkovou cenu** nebo **množství** prodeje dlouhodobého majetku.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="e5b1e-113">Zvolte sekci **Podrobnosti řádku**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="e5b1e-114">Zadejte nebo vyberte hodnotu v poli **Dlouhodobý majetek**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="e5b1e-115">Zvolte **Zaúčtovat**.</span><span class="sxs-lookup"><span data-stu-id="e5b1e-115">Select **Post**.</span></span>
