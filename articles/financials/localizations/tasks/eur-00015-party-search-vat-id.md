# <a name="eur-00015-party-search-using-vat-id"></a>EUR-00015 Vyhledání strany pomocí DIČ

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tato procedura ukazuje, jak dokončit vyhledávání strany pomocí ID registrace. Před provedením této procedury musíte nastavit DIČ a zadat libovolná DIČ pro dodavatele, zákazníky nebo právnické osoby.

Postup se vztahuje na všechny evropské země/oblasti. Procedura byla vytvořena za použití ukázkových dat společnosti DEMF s primární adresou právnické osoby v Německu. Tato procedura je určena pro manažera závazků nebo manažera pohledávek. Tato procedura je určena pro funkci, která byla přidána do aplikace Dynamics 365 for Operations verze 1611.

1. Přejděte do nabídky Správa organizace > Globální adresář > Globální adresář.
2. Klikněte na Vyhledávání registračního ID.
3. Klepněte na možnost Přidat.
4. V poli Typ registrace zadejte nebo vyberte hodnotu.
    * Například pokud jste chtěli najít strany s ID registrace typu DIČ, vyberte DIČ.  
5. V poli Země/oblast zadejte nebo vyberte hodnotu.
    * Zadejte například DEU.  
6. Zadejte hodnotu do pole Číslo registrace.
7. Klepněte na tlačítko Najít.
    * Zobrazí se všechny strany s tímto ID registrace.
