# <a name="regulatory-updates"></a>Povinné aktualizace

Doby dodání se mohou změnit a plánované funkce mohou být změněny nebo nemusí být vydány. Další informace uvádějí [Zásady společnosti Microsoft](https://go.microsoft.com/fwlink/p/?linkid=2007332).
