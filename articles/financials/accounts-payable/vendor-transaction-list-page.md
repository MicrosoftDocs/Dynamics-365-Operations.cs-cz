    <span data-ttu-id="9eda9-111">Otevře se dialogové okno **Zobrazit vyrovnání** a zobrazí vybrané transakce a všechny doklady, proti kterým probíhá vyrovnání.</span><span class="sxs-lookup"><span data-stu-id="9eda9-111">The **View settlements** dialog box opens and shows the selected transaction and all documents that are settled against it.</span></span> <span data-ttu-id="9eda9-112">Toto dialogové okno připomíná zobrazení historie vyrovnání, ale obsahuje všechny související dokumenty.</span><span class="sxs-lookup"><span data-stu-id="9eda9-112">This dialog box resembles the settlement history view, but it includes all related documents.</span></span>
