-   <span data-ttu-id="99c4d-110">**Automatické vyrovnání** – nastavte možnost na hodnotu **Ano**, má-li se transakce automaticky vyrovnat oproti otevřeným transakcím při zaúčtování.</span><span class="sxs-lookup"><span data-stu-id="99c4d-110">**Automatic settlement** – Set this option to **Yes** if a transaction should be settled automatically against other open transactions when it is posted.</span></span> <span data-ttu-id="99c4d-111">Je-li tato možnost je nastavena na hodnotu **Ne**, uživatelé mohou transakce vyrovnat ručně při zadávání plateb nebo později pomocí stránky **Vyrovnat transakce**.</span><span class="sxs-lookup"><span data-stu-id="99c4d-111">If this option is set to **No**, users can manually settle transactions when they enter payments, or later, by using the **Settle transactions** page.</span></span>
