# <a name="configure-base-entities"></a>Konfigurovat základních entit 

[!include[banner](../includes/banner.md)]


Nákladové účetnictví je nezávislý modul vyžadující, aby byly následující základní entity správně nakonfigurovány v aplikaci Dynamics 365 for Finance and Operations, Enterprise Edition.

-  Měna
-  Směnný kurz
-  Fiskální kalendář  

Před použitím modulu Nákladové účetnictví je třeba definovat pro vaši společnost měnu, směnný kurz a fiskální kalendář. 

Další informace o těchto entitách získáte v kurzu elektronického vzdělávání [Modul 2 Nastavení měny a směnného kurzu](https://mbspartner.microsoft.com/AX/CourseModules/1215) (vyžaduje účet CustomerSource).
