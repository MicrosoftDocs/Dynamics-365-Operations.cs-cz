<span data-ttu-id="11a5a-104">Po spuštění dotazu pro zobrazení transakcí v hlavní knize nebo zásobování a zdrojích může dotaz vrátit příliš mnoho informací.</span><span class="sxs-lookup"><span data-stu-id="11a5a-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="11a5a-105">Můžete tedy vyhledávání zúžit.</span><span class="sxs-lookup"><span data-stu-id="11a5a-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="11a5a-106">Chcete-li získat požadované informace, můžete procházet podrobnostmi transakce, chcete-li zobrazit sestavu výdajů, na které se nacházela transakce.</span><span class="sxs-lookup"><span data-stu-id="11a5a-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="11a5a-107">Pro sestavy výdajů je možné přejít z transakce na původní sestavu, která je uložena v modulu Správa výdajů.</span><span class="sxs-lookup"><span data-stu-id="11a5a-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="11a5a-108">Kromě celkové částky výdajů je možné zobrazit rozepsané transakce, transakce, které mají nárok na vratku DPH, a účtenky, které jsou připojeny k výdajům.</span><span class="sxs-lookup"><span data-stu-id="11a5a-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="11a5a-109">Chcete-li přejít k podrobnostem původní sestavy výdajů, na které se nacházela transakce, zvolte transakci a poté zvolte **Původní dokument**.</span><span class="sxs-lookup"><span data-stu-id="11a5a-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>
