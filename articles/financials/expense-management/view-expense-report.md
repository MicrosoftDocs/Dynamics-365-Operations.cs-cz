# <a name="view-an-expense-report"></a><span data-ttu-id="543c2-103">Zobrazit sestavu výdajů</span><span class="sxs-lookup"><span data-stu-id="543c2-103">View an expense report</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="543c2-104">Po spuštění dotazu pro zobrazení transakcí v hlavní knize nebo zásobování a zdrojích může dotaz vrátit příliš mnoho informací.</span><span class="sxs-lookup"><span data-stu-id="543c2-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="543c2-105">Můžete tedy vyhledávání zúžit.</span><span class="sxs-lookup"><span data-stu-id="543c2-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="543c2-106">Chcete-li získat požadované informace, můžete procházet podrobnostmi transakce, chcete-li zobrazit sestavu výdajů, na které se nacházela transakce.</span><span class="sxs-lookup"><span data-stu-id="543c2-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="543c2-107">Pro sestavy výdajů je možné přejít z transakce na původní sestavu, která je uložena v modulu Správa výdajů.</span><span class="sxs-lookup"><span data-stu-id="543c2-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="543c2-108">Kromě celkové částky výdajů je možné zobrazit rozepsané transakce, transakce, které mají nárok na vratku DPH, a účtenky, které jsou připojeny k výdajům.</span><span class="sxs-lookup"><span data-stu-id="543c2-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="543c2-109">Chcete-li přejít k podrobnostem původní sestavy výdajů, na které se nacházela transakce, zvolte transakci a poté zvolte **Původní dokument**.</span><span class="sxs-lookup"><span data-stu-id="543c2-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="543c2-110">Při zobrazení jednotlivých transakcí můžete také zobrazit další data, která jsou připojena k transakci.</span><span class="sxs-lookup"><span data-stu-id="543c2-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="543c2-111">Například u rozepsaného výdaje za hotel můžete zobrazit podrobný seznam výdajů, jako jsou internet, telefonáty a pokojová služba.</span><span class="sxs-lookup"><span data-stu-id="543c2-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
