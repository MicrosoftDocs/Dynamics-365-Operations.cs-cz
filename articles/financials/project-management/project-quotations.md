# <a name="project-quotations"></a><span data-ttu-id="10697-104">Nabídky projektu</span><span class="sxs-lookup"><span data-stu-id="10697-104">Project quotations</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="10697-105">Tento článek představuje koncept nabídky projektu, který můžete použít k předložení atraktivní nabídky odběrateli v rámci prvního kroku fáze projektu.</span><span class="sxs-lookup"><span data-stu-id="10697-105">This article introduces the concept of project quotations, which you can use to make an attractive offer to a customer as the first step of the project phase.</span></span> <span data-ttu-id="10697-106">Nabídka projektu může obsahovat položky a služby, základní kontaktní údaje, zvláštní obchodní ujednání a slevy, případně daně a příplatky.</span><span class="sxs-lookup"><span data-stu-id="10697-106">A project quotation might include the items and services that are quoted, basic contact information, special trade agreements and discounts, and possible taxes and surcharges.</span></span> 

<span data-ttu-id="10697-107">Schopnost sledovat, kontrolovat a řídit příležitosti projektových nabídek a objednávek přestavuje důležitou část projektového řízení.</span><span class="sxs-lookup"><span data-stu-id="10697-107">The ability to monitor, review, and control the pipeline of project quotations and orders is an important part of project management.</span></span> <span data-ttu-id="10697-108">Různé nástroje v aplikaci Microsoft Dynamics 365 for Finance and Operations vám mohou s těmito úkoly pomoci.</span><span class="sxs-lookup"><span data-stu-id="10697-108">Various tools in Microsoft Dynamics 365 for Finance and Operations can help with these tasks.</span></span> <span data-ttu-id="10697-109">Například správné definice referenčních dat (typy nabídek, původ nabídky, prognóza a pravděpodobnost nabídky) pomáhají analyzovat příležitosti.</span><span class="sxs-lookup"><span data-stu-id="10697-109">For example, correct reference data definitions (quotation types, quotation origin, and prognosis and probability) help you analyze the pipeline.</span></span> <span data-ttu-id="10697-110">Tyto nástroje lze využít k uspořádání důvodů, proč byla nabídka projektu úspěšná nebo neúspěšná, a ke zjištění potenciální hodnoty nabídky.</span><span class="sxs-lookup"><span data-stu-id="10697-110">You can use these tools to categorize the reasons why a project quotation was won or lost, and to determine the potential value of a quotation.</span></span> 

<span data-ttu-id="10697-111">Zadejte do nabídky projektu služby, základní kontaktní údaje, zvláštní obchodní ujednání a slevy, případně odhadované daně a příplatky pro projekt.</span><span class="sxs-lookup"><span data-stu-id="10697-111">In a project quotation, you enter the services, basic contact information, special trade agreements and discounts, and estimated taxes and surcharges for a project.</span></span> <span data-ttu-id="10697-112">Můžete také vybrat aktivity nebo úlohy projektu a vytvořit hierarchii a dílčí úkoly.</span><span class="sxs-lookup"><span data-stu-id="10697-112">You can also select the activities or tasks for a project, and create a hierarchy of tasks and subtasks.</span></span> <span data-ttu-id="10697-113">Pro každou aktivitu můžete zadat podrobnosti o načasování a době trvání aktivity a také o dovednostech a zkušenostech, které jsou požadovány pro pracovníky, kteří provádějí aktivitu.</span><span class="sxs-lookup"><span data-stu-id="10697-113">For each activity, you can enter details about the timing and duration of the activity, and about the skills and experience that are required for workers who perform the activity.</span></span> 

<span data-ttu-id="10697-114">Nabídka projektu je nezávazný odhad práce, která musí být provedena.</span><span class="sxs-lookup"><span data-stu-id="10697-114">The project quotation is a non-binding estimate of the work that must be performed.</span></span> <span data-ttu-id="10697-115">Pokud jsou ale informace zkopírovány do nabídky projektu, který je přidružený k projektové smlouvě, stanou se tyto informace součástí závazné smlouvy mezi dvěma stranami.</span><span class="sxs-lookup"><span data-stu-id="10697-115">However, when the information in the quotation is copied to a project that is associated with a project contract, that information becomes part of a binding agreement between two parties.</span></span> 

<span data-ttu-id="10697-116">Pokud odběratel schválí nabídku projektu, můžete zkopírovat informace do nabídky projektu pro projekt.</span><span class="sxs-lookup"><span data-stu-id="10697-116">If the customer approves the project quotation, you can copy the information in the project quotation to a project.</span></span> <span data-ttu-id="10697-117">Můžete také současně zkopírovat informace o nabídce projektu do prognózy projektu.</span><span class="sxs-lookup"><span data-stu-id="10697-117">You can also copy the project quotation information to a project forecast at the same time.</span></span>
