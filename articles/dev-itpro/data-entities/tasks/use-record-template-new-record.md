# <a name="use-record-template-to-create-a-new-record"></a>Použití šablony záznamu pro vytvoření nového záznamu

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tento postup ukazuje, jak použít dříve definovanou šablonu záznamu pro vytvoření nového záznamu. Pro dokončení tohoto postupu musíte nejprve dokončit postup "Vytvoření šablony záznamů pro usnadnění vkládání dat".



Tento postup používá společnost USMF.

1. Přejděte do části Dlouhodobý majetek > Dlouhodobý majetek > Dlouhodobý majetek.
2. Klikněte na položku Nová.
    * Budete vyzváni k výběru šablony. Vyberte takovou, která odpovídá vaší obchodní potřebě.  
3. Vyhledejte na seznamu požadovaný záznam a vyberte ho.
4. Klikněte na tlačítko OK.
