# <a name="set-up-i-9-document-types"></a>Nastavení typů dokumentů I-9

[!include[task guide banner](../../../includes/task-guide-banner.md)]

Tento postup popisuje, jak zobrazit a nastavit typy dokumentů, které se používají pro ověření I-9. Předtím, než nastavíte typů dokumentů I-9, musíte také nastavit vydávající úřady a typy identifikace. K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF, která zahrnuje příklady agentur výdeje a typy identifikace, které jsou nutné k dokončení kroků.

1. Přejděte do nabídky Lidské zdroje > Pracovníci > I-9 > Typy dokumentů I-9.
2. Klikněte na položku Nová.
3. Zadejte hodnotu do pole Typ dokumentu I-9.
    * Příklad: ID školy.  
4. Vyberte typ identifikace.  Příklad: ID školy
    * Příklady typů identifikace zahrnují číslo sociálního pojištění (ČSP), číslo víza, ID pasu nebo řidičského průkazu.  
5. Vyberte seznam dokumentů I-9 použitý pro daný typ dokumentu.
    * Jako součást procesu I-9 musí zaměstnanci poskytnout zaměstnavateli dokumentaci uvádějící jejich totožnost a pracovní povolení. Web US Citizenship and Immigration Services obsahuje informace o tom, které dokumenty mohou být použity, a do jakého seznamu patří.  http://www.uscis.gov  
6. Do pole Formulář zadejte pro typ dokumentu číslo oficiálního dokumentu. Příklad: ID školy.
    * Na webu US Citizenship and Immigration Services lze najít čísla oficiálního formuláře.  http://www.uscis.gov  
7. Zaškrtněte nebo zrušte zaškrtnutí políčka Aktivní.
8. V poli Ukončit platnost nastavte datum na „27-10-2019“.
    * Datum konce platnosti je volitelné.  
9. Vyberte agenturu vydávající daný typ dokumentu. Příklad: Provincie/území
10. Klikněte na položku Uložit.
