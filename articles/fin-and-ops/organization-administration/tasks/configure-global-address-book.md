1. <span data-ttu-id="09eef-107">V navigačním podokně přejděte na **Moduly > Správa organizace > Globální adresář > Parametry globálního adresáře**.</span><span class="sxs-lookup"><span data-stu-id="09eef-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>

9. <span data-ttu-id="09eef-115">Vyberte možnost v poli **Dodavatel**.</span><span class="sxs-lookup"><span data-stu-id="09eef-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="09eef-116">Vyberte možnost v poli **Potenciální zákazník**.</span><span class="sxs-lookup"><span data-stu-id="09eef-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="09eef-117">Vyberte možnost v poli **Konkurent**.</span><span class="sxs-lookup"><span data-stu-id="09eef-117">In the **Competitor** field, select an option.</span></span>
