# <a name="change-the-banners-or-logo-images-for-legal-entities"></a><span data-ttu-id="33651-103">Změna bannerů nebo loga pro právnické osoby</span><span class="sxs-lookup"><span data-stu-id="33651-103">Change the banners or logo images for legal entities</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="33651-104">V následující proceduře je uveden postup, pomocí kterého mohou správci systému aktualizovat proužkovou reklamu nebo obrázek loga, které se zobrazí pro právnickou osobu.</span><span class="sxs-lookup"><span data-stu-id="33651-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="33651-105">K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="33651-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="33651-106">Přejděte do části na Správa organizace > Organizace > Právnické osoby.</span><span class="sxs-lookup"><span data-stu-id="33651-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="33651-107">Na seznamu vlevo vyberte právnickou osobu, pro kterou chcete aktualizovat proužkovou reklamu nebo logo.</span><span class="sxs-lookup"><span data-stu-id="33651-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="33651-108">Pokud je již vybrána, pokračujte dalším krokem.</span><span class="sxs-lookup"><span data-stu-id="33651-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="33651-109">Rozbalte sekci Obrázky.</span><span class="sxs-lookup"><span data-stu-id="33651-109">Expand the Images section.</span></span> <span data-ttu-id="33651-110">Pokud je již rozbalena, pokračujte dalším krokem.</span><span class="sxs-lookup"><span data-stu-id="33651-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="33651-111">Klikněte na možnost Upravit.</span><span class="sxs-lookup"><span data-stu-id="33651-111">Click Edit.</span></span>
5. <span data-ttu-id="33651-112">V poli Typ obrázku řídicího panelu společnosti, zaškrtněte políčko „Proužková reklama“.</span><span class="sxs-lookup"><span data-stu-id="33651-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="33651-113">Klikněte na tlačítko Změnit.</span><span class="sxs-lookup"><span data-stu-id="33651-113">Click Change.</span></span>
    * <span data-ttu-id="33651-114">Tip: Ideální rozlišení pro obrázek s proužkovou reklamou je 1920 x 281 pixelů.</span><span class="sxs-lookup"><span data-stu-id="33651-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="33651-115">Ideální šířka pro obrázek loga je 350 pixelů.</span><span class="sxs-lookup"><span data-stu-id="33651-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="33651-116">Po dokončení odeslání byste měli v části Obrázek řídicího panelu vidět svoji novou proužkovou reklamu nebo obrázek loga.</span><span class="sxs-lookup"><span data-stu-id="33651-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="33651-117">Klikněte na položku Uložit.</span><span class="sxs-lookup"><span data-stu-id="33651-117">Click Save.</span></span>
8. <span data-ttu-id="33651-118">Přejděte na možnost Společné > Společné > Výchozí řídicí panel.</span><span class="sxs-lookup"><span data-stu-id="33651-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="33651-119">Měli byste na řídícím panelu vidět novou proužkovou reklamu nebo obrázek loga.</span><span class="sxs-lookup"><span data-stu-id="33651-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="33651-120">Může být nutné změnit vaší společnost pomocí pole pro výběr společnosti na tu, pro kterou jste odeslali proužkovou reklamu.</span><span class="sxs-lookup"><span data-stu-id="33651-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
