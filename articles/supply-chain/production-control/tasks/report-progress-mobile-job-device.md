# <a name="report-progress-on-a-mobile-job-device"></a><span data-ttu-id="31680-103">Hlášení průběhu na mobilním pracovním zařízení</span><span class="sxs-lookup"><span data-stu-id="31680-103">Report progress on a mobile job device</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="31680-104">Tento postup popisuje spuštění a hlášení průběhu výrobní úlohy v registračním formuláři úloh zařízení.</span><span class="sxs-lookup"><span data-stu-id="31680-104">This procedure shows you how to start and report progress on a production job in the job device registration form.</span></span>



<span data-ttu-id="31680-105">Aby bylo možné spuštění tohoto postupu, musíte mít roli správce systému nebo operátora stroje přidruženou k uživatelskému účtu.</span><span class="sxs-lookup"><span data-stu-id="31680-105">To be able to run this procedure you must have the System administrator or Machine Operator role associated with the user account.</span></span>
