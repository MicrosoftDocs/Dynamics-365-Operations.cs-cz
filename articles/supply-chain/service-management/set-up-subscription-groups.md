# <a name="set-up-subscription-groups"></a><span data-ttu-id="33b71-103">Nastavit skupiny předplatného</span><span class="sxs-lookup"><span data-stu-id="33b71-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="33b71-104">Klikněte na uzel **Řízení služeb** \> **Nastavení** \> **Servisní zakázky** \> **Skupiny předplatného**.</span><span class="sxs-lookup"><span data-stu-id="33b71-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="33b71-105">Stisknutím kláves CTRL+N nebo klepnutím na možnost **Nový** na panelu nástrojů vytvořte novou skupinu předplatného.</span><span class="sxs-lookup"><span data-stu-id="33b71-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="33b71-106">V poli **skupina předplatného** zadejte název nové skupiny předplatného.</span><span class="sxs-lookup"><span data-stu-id="33b71-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="33b71-107">V poli **fakturace** vyberte četnost fakturace předplatného.</span><span class="sxs-lookup"><span data-stu-id="33b71-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="33b71-108">Četnost fakturace je určena kódem období, který je nastaven ve formuláři **Typy období**.</span><span class="sxs-lookup"><span data-stu-id="33b71-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="33b71-109">Pokud chcete časově rozlišit výnosy u transakcí předplatného, zaškrtněte políčko **Časově rozlišené výnosy** a v poli **Časové rozlišení** vyberte požadovanou četnost.</span><span class="sxs-lookup"><span data-stu-id="33b71-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="33b71-110">Stejně jako četnost fakturace je i četnost časového rozlišení založena na kódu období nastaveném ve formuláři **Období**.</span><span class="sxs-lookup"><span data-stu-id="33b71-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
