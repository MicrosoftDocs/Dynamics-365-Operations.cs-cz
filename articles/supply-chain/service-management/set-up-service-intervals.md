[!include [banner](../includes/banner.md)]

<span data-ttu-id="e90f4-104">Servisní interval určuje, jak často jsou pro řádky servisní smlouvy vytvářeny řádky servisní zakázky při automatickém vytváření servisních zakázek.</span><span class="sxs-lookup"><span data-stu-id="e90f4-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

2. <span data-ttu-id="e90f4-106">Vytvořte nový interval servisu.</span><span class="sxs-lookup"><span data-stu-id="e90f4-106">Create a new service interval.</span></span>
3. <span data-ttu-id="e90f4-107">Zadejte ID a popis intervalu servisu.</span><span class="sxs-lookup"><span data-stu-id="e90f4-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="e90f4-108">Rozsah vyberte v poli **Rozsah**.</span><span class="sxs-lookup"><span data-stu-id="e90f4-108">In the **Range** field, select the range.</span></span>
