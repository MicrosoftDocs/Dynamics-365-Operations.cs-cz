<span data-ttu-id="dba99-107">Můžete také nastavit kód příčiny vysvětlující, proč bylo zboží vráceno .</span><span class="sxs-lookup"><span data-stu-id="dba99-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="dba99-108">Další informace o kódech důvodů naleznete v tématu [nastavení kódů důvodů vrácení](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="dba99-108">For more information about reason codes, see [Set up return reason code](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="dba99-109">Klepněte na tlačítko **Prodej a marketing** \> **nastavení** \> **prodejní objednávky** \> **vrácení** \> **Kódy likvidace**.</span><span class="sxs-lookup"><span data-stu-id="dba99-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>
