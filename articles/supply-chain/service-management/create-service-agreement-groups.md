# <a name="create-service-agreement-groups"></a><span data-ttu-id="0225e-103">Vytvoření skupin servisních smluv</span><span class="sxs-lookup"><span data-stu-id="0225e-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="0225e-104">Skupiny servisních smluv jsou užitečné při řazení a filtrování servisních smluv.</span><span class="sxs-lookup"><span data-stu-id="0225e-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="0225e-105">Klikněte na **Správa servisu** \> **Nastavení** \> **Servisní smlouvy** \> **Skupiny servisních smluv**.</span><span class="sxs-lookup"><span data-stu-id="0225e-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="0225e-106">Stisknutím kombinace kláves **Alt+N** vytvořte novou skupinu servisních smluv.</span><span class="sxs-lookup"><span data-stu-id="0225e-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="0225e-107">V poli **Skupina servisních smluv** zadejte ID skupiny servisních smluv.</span><span class="sxs-lookup"><span data-stu-id="0225e-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="0225e-108">Do pole **Popis** zadejte popis nové skupiny servisních smluv.</span><span class="sxs-lookup"><span data-stu-id="0225e-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="0225e-109">Uložte změny zavřením stránky.</span><span class="sxs-lookup"><span data-stu-id="0225e-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="0225e-110">Jestliže je skupina servisních smluv definována v poli **Skupina servisních smluv** na stránce **Parametry správy servisu**, program navrhne tuto skupinu pro všechny nově vytvářené servisní smlouvy.</span><span class="sxs-lookup"><span data-stu-id="0225e-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
