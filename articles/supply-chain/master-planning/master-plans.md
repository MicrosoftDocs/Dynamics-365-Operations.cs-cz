# <a name="master-plans-overview"></a><span data-ttu-id="a1896-103">Přehled hlavních plánů</span><span class="sxs-lookup"><span data-stu-id="a1896-103">Master plans overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a1896-104">Používání různých hlavních plánů na podporu každodenního provozu své společnosti, simulovat různé strategie plánování,které chcete monitorovat a zavádět podnikovou pravidla, týkající se například zásad interní výkonnosti nebo spokojenosti zákazníka.</span><span class="sxs-lookup"><span data-stu-id="a1896-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="a1896-105">Hlavní plány můžete nakonfigurovat na stránce **Hlavní plány**.</span><span class="sxs-lookup"><span data-stu-id="a1896-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="a1896-106">Existují dva typy plánů:</span><span class="sxs-lookup"><span data-stu-id="a1896-106">There are two types of plans:</span></span>
-   <span data-ttu-id="a1896-107">**Statický plán** – hlavní plánovací výpočet používá aktuální data pro vygenerování plánu čistých požadavků.</span><span class="sxs-lookup"><span data-stu-id="a1896-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="a1896-108">Tento plán zůstává nezměněný do následujícího spuštění hlavního plánování.</span><span class="sxs-lookup"><span data-stu-id="a1896-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="a1896-109">Jedná se o operační plán, který může používat různý personál společnosti, například nákupčí nebo plánovač výroby a zakládat na něm svá rozhodnutí a provádět každodenní úkoly a činnosti.</span><span class="sxs-lookup"><span data-stu-id="a1896-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="a1896-110">**Dynamický plán** – Tento plán začíná se stejným plánem čistých požadavků, který byl vygenerován hlavním plánováním.</span><span class="sxs-lookup"><span data-stu-id="a1896-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="a1896-111">Dynamický plán však můžete aktualizovat vždy při změně hlavních dat.</span><span class="sxs-lookup"><span data-stu-id="a1896-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="a1896-112">To může být například při vytvoření nové prodejní objednávky.</span><span class="sxs-lookup"><span data-stu-id="a1896-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="a1896-113">To vám umožní sledovat síť změnových příkazů dostupnost položky bez narušení statického plánu, který používají ostatní pracovníci pro své pracovní procesy.</span><span class="sxs-lookup"><span data-stu-id="a1896-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="a1896-114">Společnost si může vybrat práci pouze s dynamickým plánem nebo používat statický i dynamický plán.</span><span class="sxs-lookup"><span data-stu-id="a1896-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="a1896-115">Kromě toho můžete nakonfigurovat jakýkoliv hlavní plán a zohlednit tak specifickou strategii nebo vyřešit problém.</span><span class="sxs-lookup"><span data-stu-id="a1896-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="a1896-116">Příklady jsou následující:</span><span class="sxs-lookup"><span data-stu-id="a1896-116">Examples are as follows:</span></span>
-   <span data-ttu-id="a1896-117">Nastavení vyšší úrovně skladu na ochranu před vyčerpáním zásob.</span><span class="sxs-lookup"><span data-stu-id="a1896-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="a1896-118">Nastavení delších pojistných dob na ochranu před nespolehlivými dodavateli.</span><span class="sxs-lookup"><span data-stu-id="a1896-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="a1896-119">Můžete také nastavit počáteční dynamický plán tak, aby se aktualizoval s novými požadavky při každém spuštění hlavního plánování.</span><span class="sxs-lookup"><span data-stu-id="a1896-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="a1896-120">Tato nastavení můžete zadat na stránce **Parametry hlavního plánování**.</span><span class="sxs-lookup"><span data-stu-id="a1896-120">You can specify these settings on the **Master planning parameters** page.</span></span>
