# <a name="set-up-an-appointment-for-a-load"></a>Naplánování události pro vytížení

[!include [banner](../../includes/banner.md)]

Tato procedura ukazuje, jak nastavit a naplánovat událostí překladiště pro vytížení. To obvykle provádí koordinátor přepravy. Tento postup můžete projít v ukázkových datech společnosti USMF.


## <a name="select-the-load"></a>Výběr vytížení
1. Přejděte do nabídky Správa přepravy > Plánování > Pracovní plocha plánování vytížení.
2. Zrušte zaškrtnutí políčka Skrýt expedované a přijaté.
3. V seznamu vyberte vytížení, která má stav Expedováno.
4. Klikněte na Přeprava.
5. Klikněte na Plánování událostí.

## <a name="create-an-appointment"></a>Vytvoření schůzky
1. Klikněte na položku Nová.
2. V poli Pravidlo událostí zadejte nebo vyberte hodnotu.
3. Klikněte na položku Uložit.
4. Klikněte na Aktualizovat stav.
5. Klikněte na položku Potvrdit.
6. Klikněte na položku Uložit.
7. Zavřete stránku.
