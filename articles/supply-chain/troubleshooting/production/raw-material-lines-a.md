# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Uživatelé mohou zrušit výběr řádků materiálu u produktů, které byly označeny jako dokončené

Číslo článku znalostní báze: 4614721

## <a name="symptoms"></a>Příznaky

Systém umožňuje uživatelům zrušit výběr řádků surovin u produktů, které byly označeny jako dokončené.

## <a name="resolution"></a>Rozlišení

Systém se chová tak, jak byl navržen. Spotřebu surovin lze upravit, dokud výrobní zakázka nedosáhne stavu *Skončil*.
