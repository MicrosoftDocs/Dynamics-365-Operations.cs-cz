# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Název prostředku se neaktualizuje po změně jména pracovníka

## <a name="symptoms"></a>Příznaky

Pokud změníte jméno pracovníka v globálním adresáři, odpovídající název zdroje se v hlavní skupině neaktualizuje.

## <a name="resolution"></a>Řešení

Odpovídající název zdroje se aktuálně neaktualizuje automaticky. Chcete-li problém vyřešit, musíte ručně aktualizovat jméno zdroje.
