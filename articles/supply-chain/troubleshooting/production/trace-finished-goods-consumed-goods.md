# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Nelze vysledovat sériová čísla hotových výrobků proti spotřebovanému zboží

## <a name="symptoms"></a>Příznaky

Sériová čísla dokončeného zboží nelze sledovat oproti sériovým číslům materiálů, které výrobní zakázka spotřebuje k výrobě dokončeného zboží.

## <a name="workaround"></a>Alternativní řešení

Řešením je vytvořit výrobní zakázky pro množství 1.
