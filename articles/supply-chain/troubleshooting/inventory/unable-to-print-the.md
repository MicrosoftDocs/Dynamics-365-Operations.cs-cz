# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Sestava výběrového seznamu nezobrazuje nové poznámky připojené k deníku

Číslo článku znalostní báze: 4611660

## <a name="symptoms"></a>Příznaky

Sestava výdejky nezobrazuje nové poznámky připojené k deníku výdejky.

## <a name="resolution"></a>Rozlišení

Sestava seznamu výdejky zobrazuje pouze poznámky, které jsou připojeny k řádku prodejní objednávky a záhlaví prodejní objednávky. Nezobrazuje žádné nové poznámky, které jsou přidány do deníku faktur.
