# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Při skenování ID registrační značky v mobilní aplikaci došlo k chybě registrační značky

## <a name="symptoms"></a>Příznaky

Když skenujete ID registrační značky v mobilní aplikaci Warehouse Management, může se zobrazit následující chybová zpráva:

> Neplatná registrační značka

## <a name="resolution"></a>Řešení

Ujistěte se, že v tabulce registračních značek existuje ID registrační značky a že položky a množství na registrační značce jsou k dispozici (jinými slovy nejsou blokovány).
