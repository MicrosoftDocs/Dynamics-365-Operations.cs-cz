# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Zásilka pro nakládku byla potvrzena, ale nejsou zaúčtovány žádné řádky

## <a name="symptoms"></a>Příznaky

Při práci s odchozími skladovými operacemi se může zobrazit následující zpráva:

> Dodávka pro náklad 1% byla potvrzena.

K dalšímu zveřejnění však nedošlo.

## <a name="resolution"></a>Řešení

To je záměrné. Potvrzení zásilky nemá vliv na odeslání. Pouze aktualizuje stav zásilky a nákladu. Vystavení musí proběhnout v samostatném procesu.
