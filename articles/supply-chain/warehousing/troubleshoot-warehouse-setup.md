# <a name="troubleshoot-warehouse-setup"></a>Odstraňování problémů s nastavením skladu

[!include [banner](../includes/banner.md)]

Toto téma popisuje, jak vyřešit běžné problémy, s nimiž se můžete setkat při nastavování skladu v Microsoft Dynamics 365 Supply Chain Management.

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>K přístupu k emulátoru aplikace mobilního zařízení nemohu použít žádnou roli kromě správce.

### <a name="issue-description"></a>Popis problému

K přístupu k emulátoru aplikace mobilního zařízení nemůžete použít žádnou roli kromě správce.

### <a name="issue-resolution"></a>Řešení problému

Emulátor aplikace pro mobilní zařízení je nastaven tak, aby fungoval pouze s účtem správce. Pro všechny účely testování a živého procesu doporučujeme použít samotnou aplikaci skladu.
