# <a name="manage-standard-cost-updates"></a><span data-ttu-id="c813e-103">Správa aktualizací standardních nákladů</span><span class="sxs-lookup"><span data-stu-id="c813e-103">Manage standard cost updates</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="c813e-104">Aktualizace dat standardních nákladů lze spravovat pomocí dvou různých metod – metody s jednou verzí nebo se dvěma verzemi.</span><span class="sxs-lookup"><span data-stu-id="c813e-104">Updates to standard cost data can be managed by using two different approaches - the one-version approach or the two-version approach.</span></span>

<span data-ttu-id="c813e-105">Metoda s jednou verzí používá jednu nákladovou verzi obsahující všechny záznamy o nákladech.</span><span class="sxs-lookup"><span data-stu-id="c813e-105">The one-version approach uses a single costing version that contains all cost records.</span></span> <span data-ttu-id="c813e-106">Tyto záznamy zahrnují původní náklady a všechny aktualizace nákladů.</span><span class="sxs-lookup"><span data-stu-id="c813e-106">These records include the original costs and all cost updates.</span></span>

<span data-ttu-id="c813e-107">Metoda se dvěma verzemi využívá jednu verzi, která obsahuje záznamy o původních nákladech, a druhou verzi, která obsahuje záznamy o všech aktualizacích nákladů.</span><span class="sxs-lookup"><span data-stu-id="c813e-107">The two-version approach uses one version that contains records of the original costs and a second version that contains records of all cost updates.</span></span> <span data-ttu-id="c813e-108">Hlavní výhodou metody se dvěma verzemi je jasné zobrazení a možnost sledování aktualizací nákladů v samostatné verzi nákladů, bez ovlivnění původní verze nákladů.</span><span class="sxs-lookup"><span data-stu-id="c813e-108">A primary advantage of the two-version approach is the clear delineation and tracking of cost updates in a separate costing version, without affecting the original costing version.</span></span> <span data-ttu-id="c813e-109">Metodu se dvěma verzemi lze použít k identifikaci více přírůstkových aktualizací, kdy každé přírůstkové aktualizaci bude odpovídat samostatná verze nákladů obsahující přírůstkové záznamy nákladů.</span><span class="sxs-lookup"><span data-stu-id="c813e-109">The two-version approach can be used to identify multiple incremental updates, where each incremental update has a separate costing version that contains the incremental cost records.</span></span>
