<span data-ttu-id="6ac7f-104">Aktualizace dat standardních nákladů lze spravovat pomocí dvou různých metod – metody s jednou verzí a se dvěma verzemi.</span><span class="sxs-lookup"><span data-stu-id="6ac7f-104">Updates to standard cost data can be managed by using two different approaches -  the one-version approach and the two-version approach.</span></span>

<span data-ttu-id="6ac7f-107">Metoda se dvěma verzemi využívá jednu verzi, která obsahuje záznamy o původních nákladech, a druhou verzi, která obsahuje záznamy o všech aktualizacích class="sxs-lookup"><span data-stu-id="6ac7f-110">**Example** The following example illustrates how the one-version and two-version approaches can be used for updating standard costs in a manufacturing updates that reflect new items or error corrections.</span></span> <span data-ttu-id="6ac7f-112">Předpokládejme, že jedna nákladová verze reprezentuje standardní náklady pro aktuální rok.</span><span 2016.</span><span class="sxs-lookup"><span data-stu-id="6ac7f-115">Additionally, it contains all routing-related cost categories and overhead calculation formulas that were known at the start of the year 2016.</span></span> <span data-ttu-id="6ac7f-116">2016-STD je původní verze ocenění.</span><span class="sxs-lookup"><span data-stu-id="6ac7f-116">2016-STD is the original costing -   <span data-ttu-id="6ac7f-117">**Metoda aktualizace údajů nákladů s použitím jedné verze** − Metoda s jednou verzí znamená, že původní nákladová verze 2016-STD bude obsahovat všechny záznamy nákladů.</span><span class="sxs-lookup"><span data-stu-id="6ac7f-117">**One-version approach to cost data updates** − In the one-version approach, the original costing version 2016-STD contains all cost records.</span></span> <span data-ttu-id="6ac7f-118">Aktualizace nákladů jsou zaznamenány do 2016-STD a jsou nastavena na stav "Nevyřízené".</span><span class="sxs-lookup"><span data-stu-id="6ac7f-118">Cost updates are recorded in 2016-STD and are set to a status of ”Pending.”</span></span> <span data-ttu-id="6ac7f-119">Nevyřízené náklady lze zadat ručně pro nově zakoupené položky nebo mohou být vypočteny pro vyráběnou položku cílem zohlednit opravy.
