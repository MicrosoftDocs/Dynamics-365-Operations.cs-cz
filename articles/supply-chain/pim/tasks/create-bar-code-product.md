[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="e49dc-106">Na domovské stránce vyberte **Údržba uvolněného produktu** .</span><span class="sxs-lookup"><span data-stu-id="e49dc-106">Select **Released product maintenance** on the homepage.</span></span>

3. <span data-ttu-id="e49dc-108">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="e49dc-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="e49dc-109">V tomto příkladu vyberte číslo položky **M0001** .</span><span class="sxs-lookup"><span data-stu-id="e49dc-109">For this example, select item number **M0001** .</span></span>

7. <span data-ttu-id="e49dc-113">V seznamu níže označte vybraný řádek, který je vytvořen.</span><span class="sxs-lookup"><span data-stu-id="e49dc-113">Mark the selected row that is created in the list below.</span></span>

10. <span data-ttu-id="e49dc-116">Do pole **Čárový kód** zadejte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="e49dc-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="e49dc-117">Zadejte číslo do pole **Množství** .</span><span class="sxs-lookup"><span data-stu-id="e49dc-117">In the **Quantity** field, enter a number.</span></span>
