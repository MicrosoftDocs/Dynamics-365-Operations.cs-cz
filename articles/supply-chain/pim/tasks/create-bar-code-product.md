# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="9b88c-103">Vytvoření čárového kódu pro produkt</span><span class="sxs-lookup"><span data-stu-id="9b88c-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="9b88c-104">Toto téma ukazuje, jak ručně vytvořit čárový kód pomocí čísla položky M0001 jako příklad.</span><span class="sxs-lookup"><span data-stu-id="9b88c-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="9b88c-105">K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="9b88c-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="9b88c-106">Na domovské stránce vyberte **Údržba uvolněného produktu**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="9b88c-107">Přejděte na **Produkty > Vydané produkty** v části **Odkazy**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="9b88c-108">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="9b88c-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="9b88c-109">V tomto příkladu vyberte číslo položky **M0001**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="9b88c-110">V podokně akcí klikněte na možnost **Spravovat sklad**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="9b88c-111">Vyberte **Čárové kódy**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="9b88c-112">Zvolte **Nové**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-112">Select **New**.</span></span>
7. <span data-ttu-id="9b88c-113">V seznamu níže označte vybraný řádek, který je vytvořen.</span><span class="sxs-lookup"><span data-stu-id="9b88c-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="9b88c-114">V poli **Nastavení čárového kódu** zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="9b88c-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="9b88c-115">V poli **Čárový kód** zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="9b88c-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="9b88c-116">Do pole **Čárový kód** zadejte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="9b88c-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="9b88c-117">Zadejte číslo do pole **Množství**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="9b88c-118">Zvolte **Uložit**.</span><span class="sxs-lookup"><span data-stu-id="9b88c-118">Select **Save**.</span></span>
13. <span data-ttu-id="9b88c-119">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="9b88c-119">Close the page.</span></span>
