# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="2ba85-103">Vytvoření čárového kódu pro produkt</span><span class="sxs-lookup"><span data-stu-id="2ba85-103">Create a bar code for a product</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="2ba85-104">Tato procedura ukazuje, jak ručně vytvořit čárový kód pomocí čísla položky M0001 jako příklad.</span><span class="sxs-lookup"><span data-stu-id="2ba85-104">This procedure shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="2ba85-105">K vytvoření tohoto postupu jsou použita ukázková data společnosti USMF.</span><span class="sxs-lookup"><span data-stu-id="2ba85-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="2ba85-106">Klikněte na možnost Údržba uvolněného produktu.</span><span class="sxs-lookup"><span data-stu-id="2ba85-106">Click Released product maintenance.</span></span>
2. <span data-ttu-id="2ba85-107">Klepněte na možnost Uvolněné produkty.</span><span class="sxs-lookup"><span data-stu-id="2ba85-107">Click Released products.</span></span>
3. <span data-ttu-id="2ba85-108">Vyhledejte na seznamu požadovaný záznam a vyberte ho.</span><span class="sxs-lookup"><span data-stu-id="2ba85-108">In the list, find and select the desired record.</span></span>
4. <span data-ttu-id="2ba85-109">V podokně akcí klikněte na možnost Spravovat sklad.</span><span class="sxs-lookup"><span data-stu-id="2ba85-109">On the Action Pane, click Manage inventory.</span></span>
5. <span data-ttu-id="2ba85-110">Klikněte na Čárové kódy.</span><span class="sxs-lookup"><span data-stu-id="2ba85-110">Click Bar codes.</span></span>
6. <span data-ttu-id="2ba85-111">Klepněte na možnost Nový.</span><span class="sxs-lookup"><span data-stu-id="2ba85-111">Click New.</span></span>
7. <span data-ttu-id="2ba85-112">Označte v seznamu vybraný řádek.</span><span class="sxs-lookup"><span data-stu-id="2ba85-112">In the list, mark the selected row.</span></span>
8. <span data-ttu-id="2ba85-113">V poli Nastavení čárového kódu zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2ba85-113">In the Barcode setup field, enter or select a value.</span></span>
9. <span data-ttu-id="2ba85-114">V poli Čárový kód zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2ba85-114">In the Bar code field, enter or select a value.</span></span>
10. <span data-ttu-id="2ba85-115">Zadejte hodnotu do pole Čárový kód.</span><span class="sxs-lookup"><span data-stu-id="2ba85-115">In the Bar code field, type a value.</span></span>
    * <span data-ttu-id="2ba85-116">Stiskněte klávesu Tab.</span><span class="sxs-lookup"><span data-stu-id="2ba85-116">Press the Tab key.</span></span>  
11. <span data-ttu-id="2ba85-117">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="2ba85-117">Close the page.</span></span>
12. <span data-ttu-id="2ba85-118">Zadejte číslo do pole Množství.</span><span class="sxs-lookup"><span data-stu-id="2ba85-118">In the Quantity field, enter a number.</span></span>
13. <span data-ttu-id="2ba85-119">Klikněte na položku Uložit.</span><span class="sxs-lookup"><span data-stu-id="2ba85-119">Click Save.</span></span>
    * <span data-ttu-id="2ba85-120">Když klepnete na tlačítko Uložit, je spuštěna kontrola čárového kódu a v takovém případě se zobrazí chybová zpráva s informací, že očekávaná kontrolní číslice je 8, ale byla nalezena 3.</span><span class="sxs-lookup"><span data-stu-id="2ba85-120">When you click Save, the barcode check is run, and in this case it will display an error stating that the expected check digit is 8, but that 3 was found.</span></span> <span data-ttu-id="2ba85-121">Ručně aktualizujte číslo čárového kódu, aby bylo na konci číslo 8.</span><span class="sxs-lookup"><span data-stu-id="2ba85-121">Manually update the barcode number so that 8 is at the end.</span></span>  
14. <span data-ttu-id="2ba85-122">V poli Čárový kód zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="2ba85-122">In the Bar code field, enter or select a value.</span></span>
15. <span data-ttu-id="2ba85-123">Zadejte hodnotu do pole Čárový kód.</span><span class="sxs-lookup"><span data-stu-id="2ba85-123">In the Bar code field, type a value.</span></span>
    * <span data-ttu-id="2ba85-124">Stiskněte klávesu Tab.</span><span class="sxs-lookup"><span data-stu-id="2ba85-124">Press the Tab key.</span></span>  
16. <span data-ttu-id="2ba85-125">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="2ba85-125">Close the page.</span></span>
17. <span data-ttu-id="2ba85-126">Klikněte na položku Uložit.</span><span class="sxs-lookup"><span data-stu-id="2ba85-126">Click Save.</span></span>
18. <span data-ttu-id="2ba85-127">Zavřete stránku.</span><span class="sxs-lookup"><span data-stu-id="2ba85-127">Close the page.</span></span>
