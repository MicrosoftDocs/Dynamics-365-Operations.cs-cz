[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="4aa34-104">Tato procedura ukazuje, jak generovat sestavu odsouhlasení daně, která je k dispozici pro islandské a norské právnické osoby.</span><span class="sxs-lookup"><span data-stu-id="4aa34-104">This procedure shows how to generate the tax reconciliation report that is available for Icelandic and Norwegian legal entities.</span></span> <span data-ttu-id="4aa34-105">Byla vytvořena za použití ukázkových dat společnosti DEMF s hodnotou země/oblasti právnické osoby aktualizovanou na Norsko.</span><span class="sxs-lookup"><span data-stu-id="4aa34-105">This task was created using the demo data company DEMF with the country/region of legal entity primary address updated to be Norway.</span></span>

1. <span data-ttu-id="4aa34-106">Přejděte na Daň > Dotazy a sestavy > Sestavy DPH > Sestava odsouhlasení daně.</span><span class="sxs-lookup"><span data-stu-id="4aa34-106">Go to Tax > Inquiries and reports > Sales tax reports > Tax reconciliation report.</span></span>
2. <span data-ttu-id="4aa34-107">V poli Z účtu zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="4aa34-107">In the From account field, enter or select a value.</span></span>
    * <span data-ttu-id="4aa34-108">Vyberte hodnotu 200140.</span><span class="sxs-lookup"><span data-stu-id="4aa34-108">Select the value 200140.</span></span>  
3. <span data-ttu-id="4aa34-109">V poli Na účet zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="4aa34-109">In the To account field, enter or select a value.</span></span>
    * <span data-ttu-id="4aa34-110">Vyberte hodnotu 600180.</span><span class="sxs-lookup"><span data-stu-id="4aa34-110">Select the value 600180.</span></span>  
4. <span data-ttu-id="4aa34-111">V poli Období vyrovnání zadejte nebo vyberte hodnotu.</span><span class="sxs-lookup"><span data-stu-id="4aa34-111">In the Settlement period field, enter or select a value.</span></span>
    * <span data-ttu-id="4aa34-112">Vyberte hodnotu Pondělí.</span><span class="sxs-lookup"><span data-stu-id="4aa34-112">Select the value Mon.</span></span>  
5. <span data-ttu-id="4aa34-113">Zadejte datum do pole Od data.</span><span class="sxs-lookup"><span data-stu-id="4aa34-113">In the From date field, enter a date.</span></span>
    * <span data-ttu-id="4aa34-114">Vyberte hodnotu 2015-11-01.</span><span class="sxs-lookup"><span data-stu-id="4aa34-114">Select the value 2015-11-01.</span></span>

7. <span data-ttu-id="4aa34-116">Klikněte na tlačítko OK.</span><span class="sxs-lookup"><span data-stu-id="4aa34-116">Click OK.</span></span>
