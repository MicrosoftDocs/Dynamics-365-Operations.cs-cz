1. <span data-ttu-id="e50cc-106">Přejděte do části Hlavní kniha > Účtová osnova > Dimenze > Výchozí šablony finanční dimenze.</span><span class="sxs-lookup"><span data-stu-id="e50cc-106">Go to General ledger > Chart of accounts > Dimensions > Financial dimension default templates.</span></span>

3. <span data-ttu-id="e50cc-108">Do pole ID šablony zadejte ID šablony.</span><span class="sxs-lookup"><span data-stu-id="e50cc-108">In the Template ID field, enter a Template ID..</span></span>

9. <span data-ttu-id="e50cc-115">Klepněte na možnost Přidat.</span><span class="sxs-lookup"><span data-stu-id="e50cc-115">Click Add.</span></span>
10. <span data-ttu-id="e50cc-116">Zadejte jiné procento, které zajistí celkové procentuální hodnoty rovnající se 100 %.</span><span class="sxs-lookup"><span data-stu-id="e50cc-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
